COLUMN

こっそり

アメブロは、ピーク時間帯だろうがかまわず、プログラムを改変したりするので、サービスがよく止まります。

普通は、テスト環境とコピー環境でテストを重ねた上で、バグを潰してから、メンテナンス日を告知して、改変したものをロールアウトするのが当たり前です。

本番環境にいきなり改変を行うから、サービスが停止するし、バックアップからロールバックできずに、ユーザーが編集中だったデータが消えてしまったりする訳です。

こういうのが続くから、ユーザー数が減って行っているのに気がつかないんでしょうかね。

10年以上たっても相変わらずダメな会社だな。

レモンウォーター

瀬戸田レモンとたっぷりの氷を入れた断熱タンブラーにウィルキンソンのレモン風味炭酸水を注ぐ。

10時を超えた頃からかなり暑くなってきました。

気分だけでも爽やかにしよう。

オフィスチェア

ニトリの安価な椅子が体重に耐えきれず、溶接部分が2度も破断して、曲がって直らなくなってしまったので、新しい椅子を買いました。

コクヨのデュオラ（T字肘掛け、ヘッドレスト付き、ブラックフレーム、ブラック樹脂脚）です。

定価11万円前後で通販や一般店舗で買うと安くても9万円ぐらいです。

卸のルートから購入したので5万円台で買えました。

さすがに業務用商品です。

変なガタつきも歪みもなく、しっかりとした感触に適度に柔らかく底付きのない座面。

私の足が短くて股が太いので、だらりと座ると若干座面が高く感じます。

でも背筋を伸ばして正しい姿勢にするとピタッと足の裏が接地して良い感じです。

やっぱり仕事には最適な椅子ですね。

呑みたい日もある

インドアでも独りでも割と気分が安らぐ人間ですが、さすがに気持ちが滅入ることもあります。

気分がハイになるようなことは出来ませんから、少しお酒を飲もうと思います。

トリスで気持ちよく酔えるかな、むしろ寂しさが募るような気がする。

ぐったり

心療内科からの帰りですが、母親の要望でしまむらにきています。

外気温計は38度を示しています。

車のエアコン全開ですが暑いです。

ぐったりしてます。

ほんとにしんどい。

40度

お墓参りに行ってきました。

朝8時頃は27度、お墓参りが終わって、街中に出てましたが、40度という温度になっていました。

家に帰ってきても33度。

自室の室温は35度。

エアコンをガンガン回していますが、32度からなかなか下がりません。

お墓参りに行ってきます

今日暑くなりそうです。

っていうか朝から暑いです。

高原なのに･･･

藪蚊（ヒトスジシマカ）とに戦いですが、お墓参りに行ってきます。

お家で涼しく過ごしましょう。

自分の体温すら暑苦しい

暑さとしんどさに耐えかねて、ベッドに横になってますが、室温が30度あるし、ベッドのマットレスから反射してくる自分の体温すら暑苦しいです。

扇風機がないと寝られないし、朝起きると脱水してて、足がつるし、毎日キツいです。

エアコンはコンプレッサーのゴゴゴって音が家全体に響くし、梅雨寒からの急激な猛暑は体に堪えます。

でもねなくちゃ。

暑いでゲス

暑すぎて、日陰で扇風機を使おうと、熱中症になりました。

頭痛がひどいので、痛み止めを飲みましたが、晩ご飯を作っている間は耐えられましたが、それ以降はまた痛みがでてきました。

この時間になって30度をきったようです。

晴れていても湿気が多くて蒸し暑いです。

スマホ復活

Sony Ericsson XPERIA X Performance SO-04Hのバッテリー交換をして動かなくなっていた件、裏蓋を押し込んだらいよいよ何も反応しなくなったので、再度バラしてみました。

バッテリーと本体回路へのコネクター部分が少し斜めになって浮いていました。

それをしっかりと押し込んで、USBケーブルを挿すと、今度はスマホの充電状態を示す画面が表示されました。

ということで、起動しなかった原因はバッテリーの接続不良でした。

裏蓋に2ミリ幅の両面テープをきっちりと貼り、裏蓋をはめ込み、ちゃんと使えるようになりました。

昨晩に注文したXPERIA 8のキャンセルは出来ないので、新しいスマホが届き次第、データをそちらに移し、XPERIA X Perfomanceは親に譲ります。

2022年にはauのガラケーは使えなくなるので、それに対応すると言うことで、納得しよう。
