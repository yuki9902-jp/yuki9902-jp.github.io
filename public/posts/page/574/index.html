ようやく

私の住んでいるところは、絶妙の標高と海からの距離なのか、今年の暑い梅雨でもわりと涼しいのです。

エルニーニョの影響で日本周辺の海水温が上がっていないので、今年の梅雨は乾燥気味。

私の家の周辺では、なかなか雨が降らないでいました。

今日は、夕方から久しぶりの雨。

日中は、ようやく梅雨らしい蒸し蒸しとした空気に覆われ、季節を感じることができました。

色々なモノにカビが生えないように気をつけなきゃいけない季節です。

久しぶりに

おはようございます。

今日はようやく「だるおも」状態から脱しました。

でも朝からお腹が痛いのですが、ある程度出し切ったら収まりました。

ということで久しぶりにちょっと元気な状態です。

今日は何してみよう。

紫陽花でも撮りに出かけましょうか。

体の調子の悪い方は、しっかりご養生してくださいませ。

元気な方は、楽しい日となるようにお祈りいたしております。

お仕事の方は無理をされないように、平穏な一日でありますように。

では、皆様、良い一日でありますように。

だるおも

最近、睡眠薬の効きすぎか、天気が悪いせいか、体調が悪いせいか、朝9時に目が覚めた後フラフラとトイレに行って、体も頭も重く、またすぐに眠気に誘われて昼まで寝てしまう。

2～3時間活動すると夕方にはまた眠くなってねてしまう。

それでも頭も体もはずっと痛く重いまま。

この体の異常なだるさはなんだろう。

夜は夜で、睡眠薬を飲まないと途中で目が覚めて、眠れなくなってしまうし。

こんな状態ではどこにも出かけられないし。

おかしいなぁ、働いていないのに…。

無印 Windows XP Proのインストール

32bitのWindows環境下でないと動作しないCanoScan 8000Fを使う必要に迫られたため、久しぶりに無印 Windows XP Professionalをインストールしてみた。

インストールしたのは、

CPU / AMD Phenom 9550 (Quad Core 2.2GHz 95W)

M/B / ASUS M3A32-Deluxe/Wi-Fi (AMD 790FX + AMD SB600)

Mem / 8GB (32bit環境下なので3.5GBまでしか使えない)

GPU / ASUS EN8600GT (nViDIA GeForce 8600GT 512MB)

HDD / WesternDigital Caviar Green 1.0TB

DVD / NEC DVDスーパーマルチドライブ

の自作PC。

さて、BIOS設定で、HDDの接続をAHCIからIDEに変更して、Windows XP ProfessionalのCD-ROMを起動。

20分もかからずWindows XP Professionalのインストールは完了。

M/Bに付属のDVD-ROMからドライバーをインストールしようとするもチップセット、サウンド、LANのドライバーはインストールできたが、USB2.0のドライバーがインストールできない。

Windows XP SP1以上が必要とのこと。

で、Windows XP SP1を適用しようと、Windows Updateにつなごうとしたが、Internet Explorer 6.0ではDNSエラーとなってしまう。

Microsoftの案内では、Internet Explorer 8.0をインストールするようにとでていたが、IE8のインストーラーが、無印Windows XP Professionalでは必要なDLLがないので、エラーを表示して終了する。

困ったので、Windows XP SP3をインストールしようとしたが、SP3は、SP1a以上が適用されていないと、インストールできない。

目眩

今日はすこぶる調子が悪いです。

朝は夢見が悪く、起きてからは目眩がします。

傾頭性目眩の症状が強く出ています。

頭を右に傾けると、大きく平衡感覚がなくなりぐるぐると回ります。

今日は安静にしなきゃならないのですね…。

自殺を思いとどまらせることも重要ですが、自殺しなくても済む世の中にしていかなければならないのです。

テレビを見ていると、アベノミクスの熱に浮かされた東京周辺のごく一部の派手な報道ばかりが垂れ流され、大多数の人口を占める地方や若年層、低所得者層が、疲弊していく様子がまったく流れてきません。

物事の表層部分だけ、綺麗な部分だけを見て、すべてを悟った気になっている人たちが多くて気持ち悪い。

企業も富裕層もメディアも何故、自分の欲を満たすことだけしか考えないのか。

SAO HF 175時間目。

アインクラッドの2周目をクリア。

キリトLv.142、アスナLv.138で、100層のラスボス12体を倒した後、再び「絶剣ユウキ」と対戦。
あえなく「絶剣ユウキ」の奥義技の一撃で負ける。

2周目のエンディングが流れた後、再び100層のラスボスからやり直す。
三度「絶剣ユウキ」に敗れる。

もう一度100層のラスボスからやり直す。
都合4度目で「絶剣ユウキ」に勝った。

3周目からようやく「絶剣ユウキ」をパーティーメンバーにできる。
で、何かしらの隠しイベントとかあるのかと思ったら、SAO HFのはじめの街、76層「アークソフィア」の街中を普通のモブキャラ達と同様「絶剣ユウキ」が歩いているorz。

3週目に入ってサチLv.128、アルゴLv.80、ユウキLv.100以外の原作登場ヒロインもゲームのモブキャラも、アスナより1だけ低いLv.137に並んでしまった。

自由にパーティー組みやすいが、アスナのトップギルド血盟騎士団副団長としての立場は…。

モブキャラの中に「ココア」というキャラがSAO IMの頃からいるが、SAO HFからはちょっと特別なモブキャラになった。
専用の台詞やキャラ育成、声優も別途アテレコされている。

しかし、親密度が上がるとアスナとの台詞のダブリがあって、表示される文字列が「分かるよ、夫婦だもん…。」という非常にクリティカルなところだった。
キリトさん、重婚は犯罪ですよ。

たぶん、フラグ管理と台詞データテーブルの読み出しにバグがあるか、変数に入っている値が間違っているのだろう。
