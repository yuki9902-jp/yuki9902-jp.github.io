Visual Studio 2010

Visual Studio 2010 Expressをフルでインストール中です。

Visual C#、Visual BASIC、Visual C++、Visual Web DeveroperのExpressエディションを、すべてインストールしています。

デスクトップPCとモバイルPCと二台ありますが、モバイルは先日インストールを終えて、C#のアプリを作成していました。

が、やっぱりモバイルでの開発は作業しづらいので、デスクトップにもインストールをすることにしました。

もともとはデスクトップで作業していたんですが、何度かWindowsのクリーンインストールをしているうちに、Visual Studioを入れるのが面倒くさくなって放置していたんですが、このたび作成欲が働き、C#の勉強を再開したので、開発環境が必要になったのです。

さて、C#で、iPhone風のデザインの時計アプリを作成していて、ウィンドーの透過やらタスクトレイのアイコン表示やらを組み込んでいます。

これのベースを作ったのが2010年だったかな。

で、設定の保存機能を追加しようと現在改良中です。

昔は INIファイルを使ってましたが、今はXMLファイルなんですね。

しかも簡単っぽいです。

発行しようとしたら証明書の有効期限がきれているようなので、再取得が必要なのかな。

この辺がよくわからないですね。

図らずも

ブロードバンドルーターを交換したら、スループットが激しく向上しました。

FTP96Mbpsスループットが謳い文句の有線ルーターをこれまで使っていました。

また無線ルーターも保有していたんですが、ルーター機能はOFFにして、使っていました。

機器の整理をするために、有線ルーターを撤去して、無線ルーターのルーター機能をONにして、ネットワーク設定をやり直したら、なんかインターネット接続が非常にスムーズになりました。

プロバイダーから割り合てられるIPアドレスも変わっているので、プロバイダー側の集約機器が更新されているところに割り当てられたのかもしれません。

これまで8Mbps程度が普通だったんですが、13Mbps程度で安定しています。

またダウンロードソフトでのセッション数が多くても新しいルーターのほうが処理が早いようです。

コンシューマー向けの製品も基本的なところで改善が図られているのかもしれません。

ここ数日はＣＡＤの勉強をしております。

ＣＡＤはＪＷＣＡＤというフリーウェアです。

かなり利用者の多いアプリケーションですが、非常に使いづらいのが特徴です。

で、画面を注視して作業するので、目が非常に疲れます。

一時期治っていた眼圧の低さもまた出てきました。

まったくピントが合いません。

乱視とは違う感じで、上下2重に大きく像がずれているので、メガネなどで補正できません。

目をしばらく休めれば治るんですが･･･。

まだしばらくＣＡＤの学習ですが、もううんざりです。

これから採用面接です。

変に緊張します。

経営者一人だけの株式会社ですが、面白い会社ならいいんですが。

果たしてどんなものやら。

確定申告だけど

確定申告に行ってきました。

一応、自営業者になるんでしょうか、正確には無職ですが…。

昨年の収入はありませんでしたから、申告は、医療費と保険税と年金の支出だけ。

簡単なものです。

市場ニーズの予測が立てられても、金に執着がないので儲けたいとも思わないのがいけないのでしょう。

欲が激しく湧く方法ってないもんでしょうか。

痛かったよぅ。

一昨日、晩ご飯を食べた後しばらくして、急激な歯痛に襲われまして、あまりの激痛に悶絶しておりました。

鎮痛剤がなかなか効かず、６錠飲んで、それでも効かず、顆粒の鎮痛剤を一包のんで、徐々に効き始めました。

それでも耐えがたい痛みで、右回りにくるくる回って、何とか痛みをごまかし、意識を保ちました。

歯槽膿漏による歯痛も痛風の痛みもどちらも耐え難いです。

しかもどちらも鎮痛剤が効きにくい。

食生活や口内の健康には気を遣っているんですが、免疫力や肝機能の弱さが如実に出てしまっています。

こればかりは生まれ持ったものなのでしょうがないですね。

リフォームをDo It Yourselfで

今日もリフォームしてました。

築４０年の我が家をあちこちと。

昨年末に風呂とトイレが壊れたのを機に、もともと商店をしていた屋根続きの別棟に風呂とトイレを増設して、浄化処理層をうめました。

もとあった風呂とトイレを撤去し、埋設で、浄化槽を接地したので町からの補助金をもらいました。

さて、物事は勢い次いでです。

元あった風呂の場所は小部屋に、トイレは倉庫に。

廊下の床下の根座を修理して、床板をフローリングに。

現在、さらについでにリビングの根座がシロアリによって全滅しているのを、床板を剥がして、修理中。

柱も数本、シロアリにやられています。

柱の修繕は大工にお願いしてしてもらうことにしました。

あとは家族で、自分たちの手で修復です。

まあ数百万円かかるところを数十万円で押さえられるし、やってみれば自然と施工技術が身につくのでおもしろいです。

ただ筋肉痛や関節痛がありますが…。

こんばんは。

今晩は、あまりの歯痛に、悶絶してました。

なかなかきついです。

今は、痛み止めが効いているようです。

ありがとう、サリドン。

はあ、もしかなうなら、歯をすべてきれいにしたいよ。

雪が

6時ごろから雪が降り出しました。

気温は1度ぐらいなので、積もる傍から解けていきます。

7時頃は解けるよりも早く降ってきたので、積もっていきました。

8時頃にはハラハラと降り積もりはしません。

日本海側には例年の倍というような雪が降っているようで、亡くなる方もあり、心配でもあります。

いろいろな名目でばら蒔いている庶民や地方に届かないお金を除雪にまわしても良いんじゃないかとも思います。
