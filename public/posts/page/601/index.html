遅れているの

今朝は寒いです。

氷点下４度とか、言っております。

訓練所は今日はお休み。

でも夕方には雨になりそうですし、今も曇っています。

南部は、ようやく梅が見頃となったようです。

昨年よりさらに遅れているようです。

写真を撮りに出かけたいところですが、C#の勉強をしましょうかね。

何か変わりましたか

まだ１年しかたっていないのですね。

まだ復興にはほど遠い状態だと、テレビから流れる映像を見るたびに思うのです。

一方で仙台市は、復興バブルだそうです。

商店街では貴金属がたくさん売れているそうです。

必要なところにお金が回らないのは、日本の常なのでしょうか。

何兆円もの補正予算は、被災地に届くこともなく、間に入った業者が持って逃げています。

絆絆と騒ぐテレビは、本当に絆になることをしていますか。

募金を呼びかけているばかりで、収益を義援金に寄付することぐらいはしているんでしょうか。

小さなメディアからは、「絆」や「がんばれ」という言葉を被災者は迷惑だと思っていると伝えています。

もう精一杯がんばっているし、毎日一生懸命なのに、何をがんばれというのでしょうか。

がんばらなくていいぐらいに、保護、補償をするのが、正しい方法じゃあないでしょうか。

ボランティアや平和集会に参加して、自己満足に陥っていないでしょうか。

今日、岡山でも平和に関する集会を寒風吹きすさぶ河原で行っていました。

露店を出していた家族は、親は十分に着込んでいましたが、子供は古着の薄手の着物で、がたがた震えていました。

そんな状態で、他人を思いやれる行動がとれていると、平和活動に参加していると、グリーンピースみたいに環境のために人間を犠牲にして攻撃するような行動して、ただただ思い上がりと自己満足に陥っている人間をみると反吐がでます。

他力本願も多いんではないですか。

責任を他人ばかりになすりつけていないでしょうか。

今の社会を作ったのは自分たちです。

環境意識、エコロジー（エコノミーを含むエコではない）、核廃絶、人権尊重、平和、安全を自分だけの都合で唱えているだけではないですか。

自分は大切ですが、他人も一緒に、皆、平和で笑顔でいられる世界を作っていかなければならないのです。

都合のよい部分だけを取り出して、きれい事を並べるのは、悪事です。

人間が抱える黒い心も認め、それでもなお人を愛していかねばならないのです。

原発推進と増税論で利益主義の抜けない企業、政治家には辟易します。

いったいいつまでどす黒い時代を生きていかねばならないのでしょう。

なぜ日本人は一様になろうとするのでしょうか。

なぜ個を認め尊重しないのでしょうか。

社会が正しいのでしょうか。

私は残酷で冷徹な日本人が嫌いです。

しかし私は日本人です。

だから日本人を愛す努力をします。

まじめで不器用な男の心を歌っております。

独善や偽善でない、他人への思いやりもあるような気がします。

タブレットPCに

iPad3が発表され、日本では3/16から発売されるようですが、いろいろと魅力感じております。

FullHDを超える高精細ディスプレイを搭載しているのが、写真の確認に便利そうです。

それ以外については、現行のWindowsタブレットと大差はないようです。

ハードウェアでiPad3にWindows勢が追いつけば、そろそろタブレットPCに移行してもよさそうかな。

Windows 8を搭載したタブレットPCが出れば、UIの操作性でiOSに引けをとらないでしょうね。

PC

電源を交換したミドルタワーPCも安定しています。

原因不明だったブルーバックや電源落ちも今のところありません。

他にコードレスマウスを使っていましたが、チャタリングが発生して結構なストレスになってきました。

マイクロスイッチを交換するしか根本的な解決はないのですが、分解するのが面倒くさいので、べつのマウスを使うことにしました。

しかしこのマウスもコードが内部断線しているようで、時々Windowsから切り離されたり、カーソルが動かなくなったりしてました。

断線位置が特定できたので、分解してコードを部分切除して半田付けしました。

断線の症状がなくなり普通に使えます。

ということでしばらくはストレスなく使えるかな。

さてノートPCを一度Windows 7 Home Editionに戻して、バックアップを作成したので、これからは簡単に環境復旧ができます。

再度Windows 8 Consumer Previewをインストールし直し、こちらでしばらく使ってみようかと思います。

Visual Studio 2012の評価版が公開され、Windows 8のUIに対応したアプリの開発が可能です。

これ以前のVisual Studioに対応したSDKも公開されています。

Windows PhoneとWindowsがWindows 8で統合されるらしいので、両方で同じアプリが使えるということになるので、開発も簡単になります。

復旧しました

ミドルタワーPCの電源ユニットを買い換え、交換して復旧しました。

４０００円ほどのもので、600Wの出力が可能です。

前は500Wでした。

今はATX 2.3になっているんですね。

さて今度は安定して使えると良いな。

PCが壊れる

メインで使っているミドルタワーPCの電源ユニットが壊れたようだ。

PCを使っていたときに、突然ブラックアウト。

ばっしゅーっていう電機部品が吹っ飛んだ音が聞こえたので、物理的な破損が起こったのはわかった。

電源ボタンを押しても復旧しない。

コンセントを抜いて、とりあえずPCをばらした。

最初はメインボードを疑っていたので、メインボードを外すべく、各パーツを外していった。

メインボードを抜くのに電源ユニットが邪魔になるので、電源ユニットを外したら、ケースに焦げた痕がついている。

ということはケース内で、オーバーロード、過電流が起こって部品が破損したっぽい。

電源ユニットのケースを開けると、大型のコンデンサが並んでいるところの付け根に焦げがみられる。

もう少しみてみるとヒューズが入っていた。

ヒューズを外してみると、やはりヒューズが切れている。

過電流があった証拠だ。

とりあえずはこれを交換すれば動くかもしれないが、過電流が起こったってことは、電源ユニット内に問題がありそうだから、このまま使用するわけにはいかない。

定格10A 250Vのヒューズなので、2500VAまでは耐えられるはずなのに。

電源ユニットを買い換えなければならないなぁ・・・。

Windows 8 Consumer Preview

Windows 8 Consumer Preview 32bitをノートパソコンにインストールしていじってみました。

インターフェイスの完成度はイマ３ぐらいでしょうか。

デスクトップＰＣやラップトップＰＣでは非常に使いづらかったです。

スマートフォンやタブレットＰＣやリビング（メディア）ＰＣとすれば使いやすいです。

通常のデスクトップがありましたが、スタートメニューがなくなっり、設定ＵＩが大型化した所為で、マウス操作では非常にちぐはぐとなり、迷います。

そして一番困ったのはシャットダウンボタンがなくなり、かなり深い階層に行ってしまったことでしょうか。

電源を常時入れっぱなしの機器には良いでしょうが、ＰＣとしては使いづらすぎます。

カーネルの完成度はWindows 7でかなり高くなっていて、さほど変わらないのかもしれません。

ＵＩはこれまでのものを自由に選ばせて欲しいところです。

さて

不採用の連絡があった。

何度目だろう。

うーん、能力や経歴には問題がないと思うんだけどなぁ。

求人数は、見かけ上多いけど、実際の採用はほとんどない。

と、ハローワークの職員が言っていた。

確かに県の労働局や商工会やら同友会などから企業に対して申し入れが多く、企業はしぶしぶ求人を出しているようだ。

あと若年者の助成金目当てで、採用しても日給月給で、稼働日数が少ないため、賃金を非常に低く抑えているものもあるようだが。

正直、雇う気もないのに試験や面接を受けるのも、書類を送るのも疲れた。

この時間もお金も馬鹿にはならない。

そろそろ起業しようかとも思う。

もともとそのつもりだったしね。

さて。

Visual Studio 2010

Visual Studio 2010 Expressをフルでインストール中です。

Visual C#、Visual BASIC、Visual C++、Visual Web DeveroperのExpressエディションを、すべてインストールしています。

デスクトップPCとモバイルPCと二台ありますが、モバイルは先日インストールを終えて、C#のアプリを作成していました。

が、やっぱりモバイルでの開発は作業しづらいので、デスクトップにもインストールをすることにしました。

もともとはデスクトップで作業していたんですが、何度かWindowsのクリーンインストールをしているうちに、Visual Studioを入れるのが面倒くさくなって放置していたんですが、このたび作成欲が働き、C#の勉強を再開したので、開発環境が必要になったのです。

さて、C#で、iPhone風のデザインの時計アプリを作成していて、ウィンドーの透過やらタスクトレイのアイコン表示やらを組み込んでいます。

これのベースを作ったのが2010年だったかな。

で、設定の保存機能を追加しようと現在改良中です。

昔は INIファイルを使ってましたが、今はXMLファイルなんですね。

しかも簡単っぽいです。

発行しようとしたら証明書の有効期限がきれているようなので、再取得が必要なのかな。

この辺がよくわからないですね。
