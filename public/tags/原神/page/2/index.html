Windows11にDualSenseを接続して原神を遊ぶ

PS5用コントローラー「DualSense™」をPC版「原神」でフル活用するための導入手順

長年愛用していたDUALSHOCK 3™のUSBコネクタが故障したため、新たにPS5純正のDualSense™を導入しました。その際の設定手順をまとめます。

0. 動作環境のクリーンアップ

既存のコントローラー関連ドライバーとの競合を避けるため、環境を整理します。

  • SCPドライバーの削除： Windows 11 Pro 64bit環境でDUALSHOCK 3™を使用するために導入していた「SCPドライバー」をアンインストールします。
  • Xboxコントローラーの整理： 接続していたXboxコントローラーを取り外し、関連するドライバーやアプリも削除します。

1. PCとDualSense™のUSB接続

  • ゲームの準備： 「原神」の公式サイトからインストーラーをダウンロードし、インストールを完了させておきます。
  • 有線接続： DualSense™をUSB 2.0以上のポートへUSBケーブルで接続します。

2. DualSense™のファームウェアアップデート

PCからDualSense™の機能を最大限引き出すため、最新のファームウェアに更新します。

  • ソニー公式サイトの「DualSense™ワイヤレスコントローラーのファームウェアをアップデートする」から「Windows PC用」をダウンロード。
  • 「Firmware updater for DualSense™」を起動し、指示に従ってアップデートを実行します。

3. 振動機能（ハプティックフィードバック）の準備

DualSense™のハプティックフィードバックは、Windows上では「オーディオデバイス」として制御されます。

  • サウンド設定： コントロールパネルの「サウンド」を開き、再生デバイスとして認識されている「Wireless Controller」を有効にします。
  • 既定のデバイスの確認： 既定の再生デバイスが「Wireless Controller」に切り替わってしまった場合は、普段お使いのサウンドカードやモニターへ設定を戻します。
  • スピーカーの構成： 「Wireless Controller」を選択して［構成］をクリック。「フルレンジスピーカー」の項目にある「フロント左と右」にチェックを入れます。

4. 原神での設定

  • 入力切替： 原神を起動後、初期状態ではキーボード操作になっているため、パイモンメニューの「設定」→「操作」から制御デバイスを「コントローラー」に変更します。
  • これにより、UI上のボタンアイコンがDualSense™準拠のものに自動で切り替わります。

番外編：Steam利用時の挙動とカスタマイズ

  • Steamを導入している場合、USB接続するだけで即座に認識されます。
  • Steamの「コントローラー設定」から、DualSense™の特徴的なRGB LEDの色を自由に変更可能です。

補足：遅延対策と安定性について

Bluetooth接続時の入力遅延を避けるため、今回はUSB Type-Cによる有線接続を選択しました。

さらに操作感を極めるため、ツール「HIDUSBF」を用いてポーリングレートを変更しています。デフォルトの4msから1ms（1000Hz）に設定することで、原神内のリズムゲーム要素などのシビアなタイミングが要求されるコンテンツでも、微細なズレを感じることなく快適にプレイできるようになりました。 ※HIDUSBFを使用すればBluetooth接続時でも1ms設定は可能ですが、接続の安定性を重視して有線運用としています。

303 Days

原神を再開始して303日が過ぎました。

冒険ランク57，世界ランク8になっています。

最近、なんとか深境螺旋12層がクリア出来るようになりました。

しかし星を全部獲得は出来ていません。

螺旋12層をクリアするために、バージョン2.8で復刻された「楓原万葉（かえではらかずは）」を引きました。それから同時に復刻された幼児爆弾魔である西風火花騎士「クレー」も。

楓原万葉（CV.島崎信長）

クレー（CV.久野美咲）

バージョン2.8限定で解放された「金リンゴ群島」で得られた原石やモラなどを楓原万葉の育成に投入しましたが、まだレベル70の突破をしただけです。

万葉用の翠緑の聖遺物、エウルア用の蒼白、血染めの聖遺物、雷電用の絶縁の聖遺物も厳選中なのでリソースが足りません。

で、現在の探索、ストーリー用のメインパーティーは、ジン、エウルア、雷電、夜蘭で、螺旋は前述4人を割り振ったパーティーを二つ作ります。

候補は、LV90のロサリア、申鶴、刻晴、ノエル、LV80超えのベネット、香菱、スクロース、バーバラ、煙緋、凝光、神里綾華ですが、LV80のメンバーは天賦も聖遺物も全然育っていません。

鍾離、楓原万葉、モナ、珊瑚宮心海もLV70ぐらいになっています。

まだLV50でしかない行秋の育成が急務ですが、正直、高身長の女性キャラにしか興味が無くモチベーションが湧かないという･･･。

250 Days later

原神を再開してから250日目になります。

先日バージョン2.7へ更新され、新キャラクター「夜蘭（イェラン）」が追加されました。

星4なのに星6相当といわれる「行秋」と似たような役割を果たしますが、夜蘭は元素爆発をするとぶっ壊れた強さを発揮します。

日本語の声優は遠藤綾さんです。マクロスフロンティアのシェリル・ノーム役で非常に好きな声優です。キャラの性能はともかく、それだけで1回目のガチャを回しました。

その後の解説動画なので、性能面が分かったので2凸（3回）まで引くことにしました。

今回は「夜蘭」を2凸、復刻の「魈（しょう）」、すり抜け2回（刻晴、モナ）と武器ガチャも回したので12万円という高額を使ってしまいました。

カードローンで分割です･･･。

上海の都市封鎖で更新が遅れたためゲームですることがないという飢餓感が高すぎて、箍が外れてしまいました。

で、現在のメインパーティーは

ジン・グンヒルド（CV：斎藤千和）

エウルア・ローレンス（CV:佐藤利奈）

雷電将軍（影）（CV:沢城みゆき）

夜蘭（イェラン）（CV:遠藤綾）

風、氷、雷、水で、元素爆発に必要なエネルギーが高めのパーティーですが、感電・凍結・超伝導・粉砕を一応は意識していますが、単純に元素爆発の威力が高いのとジンの瞬時の回復で、突破力も安定感も非常に高いです。

170 Days later

原神を再開して170日を過ぎました。

バージョン2.5の後半のピックアップガチャは雷電将軍と珊瑚宮心海です。

再開したときのピックアップガチャが珊瑚宮心海で、無課金でゲットしていたので、今回は雷電将軍を2凸までしました。諭吉5枚と引き換えに・・・。

雷電将軍（影）（CV沢城みゆき）

雷元素のサブアタッカー兼バッファーです。通常攻撃力は少し低いですが、元素爆発してからは敵の防御力60%を無視してダメージを与えるので結構強いです。

レベル90になって、モチーフ武器の「草薙の稲光」と聖遺物「絶縁」4セットで元素チャージ効率を高めにしておいて、八重神子の元素スキル、ジン、エウルアと合わせて元素爆発を放つと大概ダメージ表記が出ずに敵が蒸発してしまいます。

ただ炎元素には弱いので、ジンの元素爆発が多めに回るようにしておいた方が良いです。

現在のパーティー編成

ジン=グンヒルド

エウルア=ローレンス

雷電影

八重神子

成人女性の炎元素キャラが出てくるのを待っているんですが、まだ先の話でしょうかね。

150 Days

原神を再スタートしてから150日目になります。

課金以外で結構ガチャを回していて天井まであと少しだったので、少し課金してガチャで甘雨（かんう）を入手。

課金以外で原石を入手するたびガチャを回し、Ver2.5にアップする直前に鍾離（しょうり）を入手。

Ver2.5になってすぐに課金して最初の天井で八重神子（やえみこ）を入手。

バージョンアップで新しいイベントなどが追加されて、原石を大量にゲットしたので細々と武器ガチャを回して、アップデート後から単発5回目で神楽の真意、その後単発15回目で盤岩結緑（ばんがんけつりょく）を入手。

甘雨（CV：上田麗奈）

鍾離（CV：前野智昭）

八重神子（CV：佐倉綾音）

Ver2.5の前半で用意されたコンテンツは消化してしまったので、後半のガチャに出てくる雷電将軍まではコツコツと育成します。

前半のガチャが比較的安く済んだので、雷電将軍の入手も予算内で出来そうです。

111 days later

原神を再開してから、111日が過ぎました。

バージョン2.4でガチャに登場したキャラは、「申鶴（しんかく）」と「雲菫（うんきん）」です。

申鶴（CV:川澄綾子）長柄武器、氷元素、レア度5

雲菫（CV:小岩井ことり）長柄武器、岩元素、レア度4

この2つを出すまでに三人の福沢諭吉が消えました。

この二キャラのバフ効果で原神のパーティー構成として、氷オンリーパーティーと岩オンリーパーティーも注目されています。

あと、ゲーム中で手に入るガチャ石で通常ガチャを回して「九条裟羅（くじょうさら）」が出ました。

九条裟羅（CV:瀬戸麻沙美）弓、雷元素、レア度4

狐と天狗あわせたような面、赤の一本歯の下駄、黒の二対の翼、団扇と持ち物で個性を盛りまくっています。

申鶴も九条裟羅も子供の前掛けのような衣装で、胸部の下側が強調されています。

申鶴はレベル70を超えたので、次は雲菫と九条裟羅を育てていきます。

Fifty Days Later

原神(Genshin Impact)を最初からやり直して50日になりました。

　冒険ランク50，世界ランク6、メインパーティーキャラのレベル80、武器レベル80に到達しました。

　脆弱樹脂というキャラの育成アイテムとの引き換えポイントが存在していて、160が所持の上限、8分で1ずつ回復していきます。1回の交換に20もしくは40を消費します。これが育成の足枷になっていて時間的余裕があっても一気に育成することが出来ません。

　フロントランナーは世界ランク8ぐらいですから、普通の人なら三ヶ月ぐらいあれば今から始めても追いつけるかもしれません。

　ハウジングシステム「塵歌壺」で、引きこもり勢には厳しいクエスト「他人の同天壺でアイテムを10回買う」を攻略サイトが提供しているフレンド募集掲示板を使って募集し、フレンド登録を幾人かにしてもらってクエストをクリアして、塵歌壺の構築進捗が一気に進みました。

　稲妻編は沢城しょ・・、ごほん、魔神である雷電将軍の中身「影（えい）」を引き籠もりから解放するのがメインクエストですが、影はめちゃ強です。一回目はともかく二回目も勝てないですが、結局はクリアできます。

　影との戦闘の前に「淑女」との戦闘があるので、ドラゴンスパインで習得できる極寒耐性の食料、氷耐性強化、炎耐性強化のアイテム、あとパーティーに回復役が居ないとかなり厳しいです。

　既出キャラの中で一番きわどいのが影だと思います。衣装は着物が腰から下はなく、レースの入っているので黒い下着だけのように見えます。戦闘時には長柄武器を胸から上へ引き出すシーンがあり、年齢制限があるプレステでの表現の限界を狙っているようです。

漸くジンをゲット - 原神(Genshin Impact)

寒暖差で体調が悪く、ひどい頭痛に悩まされ、痛み止めを服用して家に引き籠もっています。

　原神で、ピックアップじゃない方のガチャを無課金で引き続け「ジン・グンヒルド」（★5、風、ヒーラー兼バフ兼ダメージディーラー）をゲットしました。

　キャラのレアリティーは★5ですが、風属性だし、元素爆発では広範囲回復なので、アタッカーには不向きです。

　何が良くて固執していたのか、キャラデザが良いのとモンド編では主要な登場人物なのにレアリティーが高くて入手困難という条件が原因です。

　1年たって入手できたのでようやく執着心が薄れました。

　稲妻編で使うにはレベルが低すぎるので、しばらくはジンを集中的に育成します。

原神(Genshin Impact)再び

農繁期が終わって、足も痛いし出掛けるのが億劫なので、マルチプラットフォームゲームの「原神（Genshin Impact）」のPC版をやってます。

　このゲームは、キャラ、武器のガチャ要素に賭博性が非常に強いので欧米では問題視されていますが、アジア圏では賭博に対する考え方緩いのでソシャゲー（ガチャゲー）の素地ができあがっているので、少し渋めのガチャとしてそれなりに受け入れられています。

　今年の初めに極寒山岳エリアのドラゴンスパインをクリアしたところから実際の寒さと併せて気分が滅入り、ゲームをアンインストールしてアカウントを削除していましたが、暇になったので最初からやり直しました。

　始めてから3週間を過ぎたあたりですが、冒険レベル44、主要キャラはレベル60、武器レベル70になりました。

　岩山エリア「璃月（リーウエ）」で「群玉閣（ぐんぎょくかく）」をメテオフォールして魔神を封印するので一回きりの探索で隅々まで回りました。

　諸島エリア「稲妻」に突入して雷電将軍のいる稲妻城の頂上までは登り切れていません。あと1階が登れない。

　稲妻に到着すると解放されるマイホームである「洞天壺」のランク上げで他人の壺の中に入って買い物をしなくてはいけませんが、これがソロプレイ派には難易度が非常に高い。今日はタイ人と思われる人がアクセスしてきたんですが、コミュニケーションが簡単な英語だけで、なかなか難しい。

　それから追加された釣り要素でつり上げた魚を生け簀にいれてみました。

　グッピーの種類が多いですが、すべての種類を飼えるわけではないようです。

　今のメインキャラは

  • 刻晴（こくせい）･･･★5、雷、片手剣、ダメージディーラー
  • 主人公キャラ（旅人、栄誉騎士、蛍）･･･★5、マルチ（風、岩、雷）、片手剣、ダメージディーラー、風の元素スキルで包囲突破
  • ロサリア･･･★4、氷、長槍、デバフ兼ダメージディーラー、元素爆発に雷を足して元素共鳴
  • ノエル･･･★4、岩、両手剣、タンク、元素スキルでシールドおよび回復、元素爆発で広範囲攻撃

です。

　美少女キャラより、美女キャラの方が好きなんですが、一番欲しいのは「ジン」（★5、風、片手剣、治癒バフ）ですが、ガチャが渋いのでなかなか出ません。

　アイテムのドロップ率と獲得条件が厳しいのでキャラ育成、武器育成にはかなり時間がかかります。

　まあ日本のゲームではないでストーリーに違和感を感じますが、日本っぽいキャラなので少々は耐えられるでしょう。
