電力不足は供給側の問題

この冬に急遽電力需要に対して供給が追いつかないと騒ぎ出したのはなぜか。

JOGMECが公表している「天然ガス・LNG最新動向 ―スポットLNG価格急騰！2020年トラブル頻発とパナマ運河制約で激変迫るLNG物流―」のページを見れば分かります。

この中で言及されている「（3）トラブルの推定原因」の「5．トラブルを大きく公表し、安定供給懸念に対する市場センチメント形成を狙った可能性」を読み落とさないように拾う必要があります。

Chinaでは豪州に対する制裁という意味不明な政策で、豪州からの石炭の荷揚げを禁止しています。

かわりにカナダやマレーシア、ロシアからの石炭の輸入量を増やしていますが、追いついていません。

LNGを急遽調達し始めたのが原因で、LNGが高騰しています。

これに引き摺られて、日本のLNG調達に影響が出てきました。

原発を停めてしまった日本の発電はLNGによる火力発電に頼る部分が大きくなっているので、厳冬の今は電力の需要に供給が追いつかなくなる可能性があります。

マスコミが言う家庭での消費電力が増えた影響、太陽光発電の不足の影響ではなく、LNGの急激な需要に対してパナマ運河の大渋滞で物流に遅延が発生し、それにより価格が高騰しており、LNGの調達が難しくなっているのが原因です。

ベースロード電源に関してはいろいろ言いたいことがあるんですが、とりあえずマスコミの嘘に騙されないように、健康に影響が出ないように適切に暖房を使って、生活に必要の無いところは節約しましょう。

一番の節電はテレビを消すことです。

Raspberry Pi 4でActive Directoryを（失敗）

LAN内にメインPC、ファイルサーバー、ノートPC（ラップトップw）があり、自分のユーザーアカウントの管理を一元化したほうが良いのか、STANDALONEで管理した方が良いのか迷い中。

そして手元には正直使い道に困っているRaspberry Pi 4Bの4GBモデルがあります。

64bit OSが動く環境でもありますが、Ubuntu 20.1 64bit Desktopの公式版をインストールしてみました。

Desktop版はRaspberry Pi 4B 4GBでは重いです。

ディスクがSDカードなのでしょうがないですね。

で、次にRaspberry Pi OSをインストールして、Active Directoryの構築を試みました。

DNSをSAMBAに内蔵されたDNSサーバーを使うように設定していきましたが、Port 135の通信が出来ずにDNSの逆引き登録が出来ません。

Portの開き方が分からず、FirewallをインストールしてPortを開ける方法を試そうとしましたが、Firewallのインストールに失敗します。

Raspberry Pi 3でActive Directoryの構築をしているサイトを参考にしていましたが、簡単にはいきません。

RaspiOSでの構築は一旦諦めます。

今は、Ubuntu 20.1 Server 64bitをSDカードに書き込みました。

今日はもう面倒なので構築はまた明日にします。

今日は何も手につかなかった

Linux Mintでリモートデスクトップ、仮想デスクトップVNC、Xサーバー、何を使った良いかよく分からず、とりあえずVNCをインストールしてみましたが、思ったように設定できずに、最適な情報を見つけられず迷走しています。

とある軍事作戦で世界中で大規模な停電が発生していて、DNSやルーティングが不安定で、調べ物をするには結構しんどいです。

自分のポリシーから大きくかけ離れた大手SNSからはすべてアカウントを削除したので、情報取得するのはなかなか大変です。

結局、ストレスだけが膨らみ何も手につきませんでした。

かつて自由の大海原だった

インターネットはかつて技術を自由に開発し、学問、知識の自由を得る大海原で、皆サーフィンしていた時代がありました。

WWWはWorld Wide Web、世界中に張り巡らせた蜘蛛の糸。

この基礎を開発した人は今日のインターネットでの雁字搦めの言論統制、共産主義者のプロパガンダ、フェイクニューズ、インフラの破壊活動、犯罪シンジケートのネットワーク、富の搾取に使われるようなことを考えていたわけではないのです。

インターネットが一般に公開された頃から使ってきましたが、10数年前から善意は悪意に押しつぶされ、徐々に自由を奪われ、文化を奪われ、言葉を奪われ、技術を奪われ、知力を奪われ、資産を奪われ、仕事を奪われ、ついにすべてを奪われました。

インターネットを捨てることで、たぶん自由を取り戻すことが出来るかもしれません。

インフラと呼ばれるぐらいに肥大化した大手インターネットサービス会社は、傲慢にして悪辣となり、絶対に犯してはならない罪を犯しました。

先人の築いた自由の大海原で荒稼ぎしてきたビックテック企業は今すぐ消え去って欲しい。

極寒

ヒーターの前にはりついてても寒いです。

室温が徐々に下がっていきます。

今、7度です。

外気温は-7度くらいでしょうか。

三年ぶりの寒さですが、やはりキツいです。

熱いお茶もあっという間に冷めて、冷たくなります。

さて布団に潜り込もうかな。

長時間は寝られないですが、暖をとるには布団の中が一番です。

ASUS ROG STRIX X570-F Gaming BIOS 3202 Beta

1/6にASUS ROG STRIX X570-F Gamingのバージョン3202のベータBIOSが公開されました。

先月末に公開された3201ベータBIOSはすぐに取り下げられていました。

同じくAM4 AGESA V2 PI 1.1.9.0のアップデートです。

AGESA V2 PI 1.1.9.0の更新内容は

1．DDR4のFCLKが1600MHzより高い場合、特に1800～2000MHzでの安定性を改善。

2．暗号化された指定のSOi3を使用したWindows 10のモダンスタンバイに対応。

3．ファンレスのX570チップセットに対応。

4．安定性の向上。

だそうです。

ベータBIOSですが、とりあず更新してみました。

不具合も別になく、PBO2の性能も変わりないようですが、設定項目が若干整理されているようです。

RAMに関しては、だいぶ前からG.SKILLのTrident Z NEO 8GB*4 3600MHz CL18を使っていたのでFCLKは1800MHzで使っていました。

Samsung Bダイなら問題は起きにくいのではないかと思います。

RAMをオーバークロックで使う場合にアップデートしても良いかもしれませんが、慎重に検討してください。

それ以外ではベータBIOSなので、人柱を覚悟しましょう。

雪が降っています

先程から日差しがなくなり、雪が積もり始めました。

薄暗い世の中です。

3D酔い

脳神経が疲れてきた夕方以降に原神をすると3D酔いになります。

Minecraftほど酔わないですが、原神もかなり酔います。

日本製のゲームだと3D酔いを防ぐための工夫や制限がきっちり決まっているので酔いにくいですが、海外の新興企業のゲームはダメですね。

で、3D酔いしながらも「白亜と黒竜」のミッションをすべてクリアしました。

ドラゴンスパインも起伏の激しいマップで、山の斜面やブリザードで視界が遮られるしキャラクターの動きにダイレクトに追従するカメラなので確実に酔います。

それから世界ランク5、冒険ランク45のキャップを解放したら、経験値がたまりすぎていて、冒険ランク50まで一気に上がり、そのキャップまで到達してしまいました。

キャラクターと武器のレベルがまだ80なので、これらを90にあげたらキャップを解放します。

無課金でいったら3ヶ月以上かかるかも･･･。

ストレスに耐えて

極寒マップで精神がガリガリ削られる原神のドラゴンスパインで、しばらく攻略を停めてました。

精神を削りながらも、今開催されている「白亜と黒竜」のイベントを第四幕のミッションを一つ残すのみまで進めました。

第四幕は「急凍樹」の強化版で中途半端なマルチプレイ対応です。

食料バフをかけるとソロでもいけますが、ソロでもメニュー画面を開いても進行が停止しないので、回復や食料バフ掛け中にダメージを受けます。

1月5日の5時には「白亜と黒竜」イベントが終了するので、あと4回ほど急凍樹を倒せば良いので、全ミッションクリア出来そうです。

ドラゴンスパインでは、★4の片手剣、★4の槍とそのレシピ、★4の両手剣のレシピが手に入ります。

★4の片手剣はシナリオを進めれば、必ず手に入ります。

★4の槍はドラゴンスパインの麓のキャンプ地でクエストを受注して、鍛造に必要なアイテムを集めれば入手できます。

★4の両手剣のレシピは、八つの石碑を発見して三つの箱を手に入れれば、入手できますが、必然的に「山に隠されし物」のクエストを受注して、ドラゴンスパインのワープポイントを全解放しなければいけません。

「山に隠されし物」はドラゴンスパインが追加されて3日目にはクリアしてたような気がします。

「緋紅玉髄」を集めてエリアのランクを上げると攻略に欠かせない極寒ゲージの増加を抑える料理のレシピが手に入るので、早めに「緋紅玉髄」を探すのが良いと思います。

明日、神経の調子が良ければイベントを完遂できるかな。

今日は

あまりの寒さと冬の吉備高原特有の昼間の薄暗さで気分はめっちゃダウナーでめまいがありますが、今日は弟家族が来たので、いろいろと料理を作りました。

おせちは食べ飽きてるだろうから、ステーキと白ご飯とお吸い物、茶碗蒸し、唐揚げとラーメン。

ステーキに添えた大蒜とバターとバジルをペースト状にした薬味が好評でした。

空いた時間で甥っ子のPCのCPUとグラボを交換しました。

CPU RYZEN 7 1700→RYZEN 7 3700X

GPU Geforce GTX 1050Ti 4GB→Geforce GTX 1060 6GB

両方併せてゲーム性能は60％ほど向上したはずです。

取り外したRYZEN 7 1700とGeforce GTX 1050Ti 4GBは私のファイルサーバーに移植しました。
