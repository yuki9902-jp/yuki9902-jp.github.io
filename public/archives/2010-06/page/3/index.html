むくみが

手がむくんでます。

運動どころか外出もせず籠もってたからかな。

やばいです。

むくんで指が動かない・・・。

物理・・・

NHK教育をみていたら、中間子の崩壊と非対称性とクォークについて講演の映像が流れてました。

私が大学に在籍していた頃は、クォークも今の六種類が見つかっていたわけではなく、予測の範囲に過ぎませんでした。

K中間子やB中間子もはっきりしてませんでしたし。

素粒子物理、宇宙物理はまだまだこれから発展する可能性があります。

1970年代の理論が実証されている段階かな。

私が学生だった頃は、理論物理学ではストリングス理論での「次元のたたみ込み」とかが流行ってたかな。

数学的な理論は先行してたけど、証明方法はさっぱり見つからない時代だったけど、いまはどうなんだろうね。

語学に躓いて、修学はあきらめたけど、私の能力は物理学に向いてんだろうか・・・。

SACDでないかな、でないよな・・・。

現在、坂本真綾さんの「everywhere」の楽曲をPCに取り込んで、192kHz/24bitにアップコンバート中。

コンプレッサが効き過ぎていてせっかくのSHMCDなのに音が定位感や奥行き感がなくなっています。

アップコンバートのロジックによっては、音量バランスが好ましくなるので、それを期待してコンバートしてます。

せっかく良い作品なのでSACDなど出してくれないかな。

SACDとか聞いたら、CDやダウンロードなんて選択肢はなくなっちゃうんだけどな。

Play For Sure

私の持っているDENON AVC-4320は、Windows VISTA / 7でのPlay For Sure を取得しています。

LANを通して、リモート再生、メディアサーバーからの再生、Webブラウザコントロールが利用できます。

これまでメディアサーバー機能、Webブラウザコントロールは使っていましたが、リモート再生は使ったことがなかったです。

Windows Media Playerで再生可能なオーディオファイルを、Windows上で選んで「リモート再生」をクリックすると、

こんな画面が出てAVアンプから音が出てきます。

オーディオファイルをストリーミングとしてAVアンプの送出し、AVアンプ側でデコードして再生するものです。

AVアンプの性能に頼るので、サウンドカードがしょぼかろうと高音質になります。

最近ではONKYOのAVアンプも対応しているようです。

Where is KMixer.sys in Windows7?

ちまたでは「カーネルミキサーで音が悪い」という話が流れてるんですが、これってKMixer.sysっていうもののことなんですが、Windows 7 / Windows VISTAには「ない」んですよね。

そもそもSound / Audio device へのアクセス方法が完全に見直されて、XP以前の構造をしていないんです。

XP以前の環境ではこのKMixer.sysを回避するためにASI4ALLを介して、deviceにアクセスしていたんですが、VISTA/7以降ではこのアクセスを禁止して、代わりに低レイテンシでのアクセスとしてWASAPIを用意したわけです。

アプリケーションからは、Windows Core Audio API を介してデバイスにアクセスします。

このAPIの中にWASAPIがあります。

このCore Audio API は、XP以前には搭載されることはありません。

実際プログラミングの見地からいうと、

VISTA / 7でサウンドを鳴らす限りKMixerを使うことはあり得ないということ。

WASAPIを使ったプログラミングをしたからといって高音質になるとはいえないこと。

ドライバの品質が音質に与える影響が大きいということ。

もっとも重要なのはサウンドカードの品質。

44.1khz/16bit から192khz/24bitにアップコンバートする時の音質は、どういったロジックによりどういった信号補間するかということです。

リアルタイムにソフトウェアでアップコンバート？

リアルタイムにハードウェアでアップコンバート？

事前にソフトウェアでアップコンバート？

どれでもいいんですよ。

結局のところアップコーンバーとするときのロジックが重要なんですよね。

もとの品質を超えた高音質はあり得ないので、どれくらいあり得ないデータを上手に作るかが、プログラマーやコンバーター設計者の腕の見せ所なのでしょう。

それはフィルターと呼ばれるものですが、単純にハイビットサンプリングしただけではギザギザがたった聞き苦しいデータになるので、ノイズをあえて乗っけて聞きやすい波形データを作るということです。

「音数が増える」とか「情報量が増える」とか、確かに増えるんですが、虚構です。

聞く方は「高音質」ではなく、「好き」か「嫌い」でしかないということを前提に考えた方が正しいです。

さて、事実を知っていればアップコンバート機能付きのDACに数百万円を払うのはばからしい話なのですが。

私もその虚構に昔踊らされました。

PS3のアップコンバート機能が流行った頃ですね。

日本では（海外にはない）技術論よりも感情論、宗教のようなオーディオ趣向があるのでそれが変な思い込みを与えてました。

日本は高度な技術屋とオカルト信者な消費者がいる不思議な国ですね。

Windows 7 64bitの再インストール

メインで使っているPCが、グラフィックボードを交換してから、起動時またはシャットダウン時にブルーバックになって落ちるので、再インストールを決行しました。

メールと音楽ファイル、ブックマーク、ゲームの保存データをバックアップしてから、Windows 7 64bitをクリーンインストール。

インストールは３０分ほどでした。

準備は大変だったけどね。

ドライバーをインストールして、再起動してもブルーバックにならず、安定。

そのあとアプリケーションをぼちぼちとインストールして、バックアップデータからメールとブックマークと音楽ファイルとゲームの保存データをリストアして、完全性を確認して、問題なし。

もともとWindows VISTAからのアップグレードだったので、いらないソフトも結構入っていたし、レジストリやフォルダ権限も新しくしました。

んで、すこぶる調子よくなりました。

壁紙をFF13のライトニングに換えたりして、利用環境を整えてます。

再インストールついでに、久しぶりにAge of Empires 3をしました。

おもしろいです。

いいゲームです。

さて目的のトラブル回避はちゃんとできました。

グラフィックボードを入れ替えるときはOSを再インストールした方がいいみたいですね。

かなり高度なOSですから、仕方ないのかな。

AX5670 512MD5-H

グラフィックボードを買い換えました。

AX5670 512MD5-H

ATI RADEON HD5670を搭載した製品です。

これまでnVIDIA GeForce8600GTを使ってましたが、A列車で行こう9をプレイするようになって、性能不足を感じてました。

AMDのPhenomと780FXを使っているのでnVIDIAよりATIのほうが都合がよいので、ATIのGPUを選びました。

気になる性能ですが、A列車ではGeForce8600GTに対して約2倍のFPSが出ています。

プレイ中の動きももたつきが少なくて快適です。

ハイエンドのカードには及びませんが、これで十分かな。

追伸

RADEON HD5670は、HDMIオーディオ出力に対応してます。

音質はHDMIなのでデジタルオーディオですのでそれなりですが、ジッターの具合は分かりません。

ただしWindows7 64bitでは、PCMの32kHz、44.1kHz、48kHzの16bitでしか出力できないようです。

あとCyberlinkのPowerDVD9もしくはPowerDVD for Blue-rayではHDMIからDTSとDolbyのビットストリーム出力が出来るようです。

PCMも48kHz 16bitまでだし、ビットストリームも必要ないので、これまで通りONKYOのSE-200PCI LTDを使います。

これならPCMの192kHz 24bit出力が可能ですから。
