Cherie Dolce

サークルKサンクスの「Cherie Dolce」、お弁当商品とかは男らしいものが多い中、このブランドにおけるデザートの展開はGoodだと思います。

最近のお気に入りは「スイートポテト」。

サークルKサンクスのHPには紹介がありません。

販売地域が限定されているのかな。

製造元は「ポテトシブヤ」。高知県の日高村本郷にある会社です。

サツマイモ全般の製品、いもけんぴなどを製造しています。

なぜこれがお気に入りか。

サツマイモのデザートが根本的に好きだからってのもありますし、保存料とか入ってなくて自然な味だから。

スイートポテトならどんぶり一杯食べられるかも。

Project μ M-SPEC R906 2nd

リヤのブレーキパッド、よく効きます。

ということは交換前は摩耗してほとんど効いてなかったってことか。

ストリート仕様で高温耐性があるものではないですが、ここぞと言うときに安心してブレーキが踏める。

フロントのパッドの交換時期もまもなくなので、同じメーカーを選んでみようと思う。

ローターもだいぶすり減ったので交換時期かな。

夏ボは、車につぎ込むことになるかな。

写真を撮っていなかった。

Pascoブランドのお気に入りの商品は非常に多いです。

ちびちび食べるつもりがおいしくて止まらず、あっと言うまに食べてしまった。。。

これはきっともちっとロールが好きなチャールズ・モンローさんも気に入るはず。

PascoのHPには紹介がなかったので地域限定かもしれませんが。

Bing - The New Search Engine by Microsoft

マイクロソフトの新しい検索エンジン「Bing」の日本語版βが公開されています。

http://www.bing.com/

トップ画面の背景が毎日変わるのと検索結果の画面が整理されて見やすいのが今のところの売りです。

本当の売りは、検索結果の最適化だそうですが、登録数が少ないこと、存在しているはずの必要な情報に行き着かないこと、などまだ最適化には程遠いのではないかと思います。

自分のサイトのURL登録をしても正しくクローラーに検出してもらえない状況です。

入り口のページに書いてある情報量やリンク数が少ないこと、robots.txt、sitemap.xmlを細かく書いていないことが原因かもしれません。

robots.txt、sitemap.xmlについては、正しく書きました。

Bingのヘルプに書かれているとおりなら、これで正しく登録されるはずなんですが。

Project μ M-SPEC R906

SUBARU IMPREZA WRX STI のリヤのブレーキパッドが、減ったので交換した。

父親が何も聞かずに、Project μ M-SPEC R906 を発注していた。

たぶん、自動車部品卸の社員で同じくSTIに乗っている人に勧められたんだと思う。

M-SPECはこの4月に新発売されたストリート向けのパッドです。

制動性能は、HPでは純正パッドより優れているとでている。

ダストも少なくなるような工夫がされている。

何も相談されずに発注されたので、少しいじけてたけど、今日になって調べてみて、すこし期待している。

ただ、リヤだけなのでわかりにくいとは思う。

「ならし」が終わったらリヤ制動を試してみて、良好ならフロントも換えてみようと思う。
