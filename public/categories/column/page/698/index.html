It rained not snow

夜半に雪が降っていたのですが、日付が変わる頃には雨に変わってました。おかげで路面が凍ることなく出勤できるのですが、でも凍結していれば、それを口実に休もうかと思ったんですけど。

なぜか先週末より、夕方になると猛烈に眠くなります。うつらうつらと言う感じではなく、眠さに耐えられそうにないので、眠さをかき消すのに色々してみますが、効果なしです。何とか、業務時間の終わりまで耐えて、一眠りします。昼寝どころではなく完全に爆睡です。

体内時計が完全に昼夜逆転しているのだと思います。またそれに加えて残業が続いているので、神経が休む暇がないようです。また、ストレス解消に趣味に使う時間を必ずとるので、必然的に寝不足になります。

なので、雪が降れば休んで、神経を休めようかと思ったのですが。

It snowed after a long absence...

日曜日の未明から降り続いた雪が、日曜日の晩までに２０ｃｍを超え積もりました。

天気予報をチェックしていなかったので、ちょっと驚きました。土曜日の夜に新年会があって家に帰るのが遅かったのですが、そのときは、雪が降ってもいませんでしたので、翌朝起きたときには、ちょっとショックでした。

日曜日に買い物に出ようかと思っていたのですが、それも取りやめて家でごろごろしてましたが、あっという間に一日が終わってしまいました。撮り貯めたアニメ番組をみて、ゲームをちょこっとして、昼寝して。

岡山市内でも積雪があり、岡山南部の山々が白い装いをしているのを始めてみました。さすがに通勤中で写真を撮れなかったのですけれど。

久々に雪景色を撮影せねば。

Self Build Up PC 2008 #04

VISTAの再インストールをするつもりでしたが、少し先送り。

Windows Live Writerをインストールしたのですが、自宅のVISTA x64ではうまく動かない状態です。

今、メーカーにサポートを依頼中なのでそれが解決するまでは、ちょっと待ちます。

Cannot I but install Vista?

やはりLANの調子がよろしくない。

またアプリケーションも動きがよろしくない。

ドライバもアプリケーションも、そしてなによりWindows VISTAのシステムファイルをクリーンな状態に戻すため、クリーンインストールせざるを得ないかな。

幸いにも、システムがインストールしてあるドライブで、設定やデータでバックアップしなければならないのはメールとIEのお気に入りとSonicStageCPぐらいなので、それぞれバックアップ方法も簡単なので、思い切ってインストールをしてみようかと思います。

ついでに、ごちゃごちゃしているドライブ構成も整理してみようかと思います。

再インストールに掛かるであろう時間は1時間ほど。今夜にでも実行して何事もなかったように安定させられれば良いのですが。

Test Windows Live Writer

blogをローカルで入力するツールがだんだんと公開され始めました。

Webベースのツールと違いWIndowsアプリケーションなので、GUI等での複雑な制御も可能です。

こういったツールがあれば原稿を記入するのが飛躍的に楽になります。

しかもすばらしいことにローカルで投稿イメージを確認できます。

で、今回試したのが、Microsoft Windows Live Writerです。

これまであったツールのように中途半端にblog(MovableType)のタグを無視して表示ではなく、Movable Typeでの出力結果をシミュレートしてくれます。

入力結果を本番環境で確かめる必要がないので、表組みや画像の入り方を投稿してから修正するなどの必要がなくなります。

blogの更新がおっくうになっている人は、試してみたら良いかも

Self Build Up PC 2008 #03

GeForce8600GTの成果は結構あるみたいです。

Windows VISTAでは、PCの性能を数値で見せてくれる機能があるのですが、それで見ると5.7という数値です。

前のGeForce6600GTx2(SLI)では、4.8ぐらいの数値だったので、かなりの性能向上です。

また、GeForce6600GTの頃は、ファンレス仕様だったのでSLIでは、熱の問題があり、後付で12cmの大型ファンを取り付けていました。

結構低い音で、意外に部屋の外で気になる音でした。

今度のGeForce8600GTは、ビデオカードとしては、比較的大型のファンが付けられています。

色々とファンを取り外すと、総量としての音圧が下がったので比較的静かになりました。

また、周波数も若干上に寄ったので、部屋の外では気にならないレベルになりました。

あと、500GBのHDDですが、さすがにSATA2です。かなり高速です。

前が160GBのATA133にSATA変換を使っていたので、体感差は明らかです。

発熱量も気にならないレベルです。

あまったHDD2本は、USB2の外付けケースに入れて外だしにしました。

かなり電源への負担が減ったようです。

ビデオカード1枚、HDD2本が減ったので当たり前でしょうか。

CPUやメモリを交換するにはマザーボードを交換しなくてはならないので、そのあたりは夏ぐらいですかね。

ビデオカードもHDDも1万円ちょっとで購入できたので、かなり安上がりでした。

2年前と比べてPCパーツの値下がり感があるので、自作派にはうれしいです。

Self Build Up PC 2008 #02

現状のPCを増強する形で、物欲を満たすことにしました。

新たにPCを購入するほどの予算もないし、ハイスペックを使い倒すほどのゲームもしないので、まあ、実際にちょっと困っているところで、HDDの容量追加とビデオカードをHDCP(コンテンツ保護のためのライセンス)に対応させることにしました。

HDDは久々にWestern Deigital にしました。

WD5000AAKSです。500GB、16MBキャッシュ、SATA2(3Gbps)、7200rpmです。

消費電力が低いので発熱量も少ないし、また比較的静かそうです。

ビデオカードは、これまで外れを引いたことはなかった玄人志向にしました。

GF8600GT-E256HW/HPです。GeForce 8600GT 256MB HDCP対応です。

購入した後で知ったのですが、メーカーのサイトのBBSには、低温時の不安定さが話題になっていました。

これは、カスを引いたかもしれません。

まだ、組付けをしていないので、不安です。

東南アジア圏で製造された製品は、低温耐性がない(コンデンサ、チップともに)ので、日本製の部品に置き換えが必要かもしれません。

東南アジアでは、最低気温が10度を下回ることなんて、まずないですからね。

玄人志向は、かつてハードコアな自社設計ボードを供給していましたが、現在ではOEM販売のみ、国内のPC市場不況ではしょうがないのかな。。。

今日はこれらをA店で購入しました。

店員さんに、似た商品が同じ山に積んであるけど間違えないように念押ししたけど、それでも商品を取り違えられて、購入後に開封してから、交換となりました。

交換時には、今度こそ取り違えないように、中身を確認してましたけど。

まあ、久しぶりに接客されて、ちょっと店員さんに花を持たせてあげようと思ったのですが、いろいろなところで勉強不足が露呈してますよ。

それは商品説明でも言えることなのですが。。。

店員さんが寄ってきて商品説明しようが、そんなことは百も承知なので全然意味をなさないのです。

で、ちょっと質問すると、案の定、質問に答えられない状況になりました。

バルク品を漁っている人(私)に話しかけちゃだめです。

店員さん、話しかけるときは、マニアかどうかをちゃんと見極めないとイタイです。

まあ、私は怒りはしないのでいいですが。

また、話は変わって、LANボード(PCIカード、オンボードとも)が認識されない現象が発生する問題ですが、Windows VISTAはWindows Update後(修正モジュールインストール後)にグラフィックカード、LAN、USB、バス周りのドライバが初期化されることがあるようです。

そのため不完全にドライバがロードされ認識されなくなるようです。

そのたびに、再度、ドライバを入れ直してやる必要があります。

バス周りのドライバ→グラフィックのドライバ→LANのドライバとインストールしてやれば、再び安定します。

このあたり、Windows VISTAのService Pack で修正されると良いのですが。

如何せん、問題を特定することが困難で、電源やM/B関係のトラブルのように見えます。

まあ、よくよく考えれば、同じ構成でWindows XPの時は発生していなかった問題なので、VISTAの不具合です。

Self Build Up PC 2008 #01

現在使用中のPCも丸2年になろうとしています。

最近、また起動時にLANボードを認識しなくなることが頻発し、またたまにVGAのドライバーもロードされないことがあります。

あまりに頻度が高いので、マザーボードのFlash BIOS周りで問題が発生しているような気がします。

すでに古いマザーボードなので、BIOSの更新も行われず(2006年7月で終わっていたような。。。)なので、もはや解決は見込めないでしょうが。

そこで、今のPCをWindows XP Professional (32bit/64bit共存環境)に戻すことにして、新たにWindows VISTA Ultimate x64環境を調達しようかと思います。

出来合のPCを購入するのは癪に障るので、やはり自作です。

自作も96年からこつこつとですから、12年を迎えることになりました。

構成は、AMDのCPU(Phenom 9600ぐらい)にしてみようかと思います。

Intel でないのは、マイノリティーを求めているからです。

嘘です。昔、IntelのCPUは高すぎて買えなかったので、未だに僻みで買わないだけです。

AMD歴もK6からですから10年以上にはなるのでしょうか。

その前はCyrix 6x86 でした。

さて、CPU、チップセット、GPUともAMDにしようかとおもいます。

nVIDIAのチップセットでは、性能を生かし切れないのと、AMDのリファレンスに近いモノならトラブルが少ないだろうと思うからです。

さて、いらないモノを売却して、資金を得ようかな。

プロジェクターとAVアンプを売れば10万円ぐらいにはなるかな。。。

PS3で動画

PS3のv2.1のファームウェアでWMVとDivxの再生が可能になりました。

WMVの再生は確認できましたが、Divxのファイルは真っ黒な画面のままになり、音も出ません。

いまいち、動画ファイルの再生要件が分からないのですが、PS3のサイトでは、Divx3.11以降と2GB以下なっています。

それ以外にも、ファイルの解像度が関係ありそうですが、そのあたりいまいち分かりません。

で、WMVですが、VistaのWMP11で全画面表示で再生するより明らかになめらかに再生されるので、完全にPS3がメインのメディアプレーヤーになっています。

Vistaマシンが非力なのもありますが、Vistaにプレインストールされているコーデックがいまいちなのは言うまでもありません。

何せ、サードパーティー製のコーデックを入れるとスムーズに再生されるから。

とりあえず、何とかPS3での再生要件をまとめて、快適な動画再生環境を構築しましょう。

A列車で行こう7

PS3のPLAYSTATION STORE でA5が販売されていたので、ちょっと懐かしさもあって購入して遊んでみると、はまる、はまる。

おもしろいね。

で、勢いに乗って最新版を購入したいと思って、調べてみるとA7が2005年に出てました。

2007年にマイナーバージョンアップしてVISTA対応になっていました。

Amazonで早速注文。

昨年末の３０日に届いたので、速攻インストールして、遊びまくりです。

ちょっとゲームの操作には慣れが必要ですが、やはりA列車。

また、はまってます。

グラフィックもフルHDに対応して、美麗です。

欲を言うなら２４ビットカラーのテクスチャーを用意してほしいのと、A5にあった3Dでの鑑賞モードの復活を望む。

今時、PCゲーム不況の中、続編がでるかは危ういですが、次作に期待します。
