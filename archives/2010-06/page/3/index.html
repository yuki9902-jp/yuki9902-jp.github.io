зү©зҗҶгғ»гғ»гғ»

NHKж•ҷиӮІгӮ’гҒҝгҒҰгҒ„гҒҹгӮүгҖҒдёӯй–“еӯҗгҒ®еҙ©еЈҠгҒЁйқһеҜҫз§°жҖ§гҒЁгӮҜгӮ©гғјгӮҜгҒ«гҒӨгҒ„гҒҰи¬ӣжј”гҒ®жҳ еғҸгҒҢжөҒгӮҢгҒҰгҒҫгҒ—гҒҹгҖӮ

з§ҒгҒҢеӨ§еӯҰгҒ«еңЁзұҚгҒ—гҒҰгҒ„гҒҹй ғгҒҜгҖҒгӮҜгӮ©гғјгӮҜгӮӮд»ҠгҒ®е…ӯзЁ®йЎһгҒҢиҰӢгҒӨгҒӢгҒЈгҒҰгҒ„гҒҹгӮҸгҒ‘гҒ§гҒҜгҒӘгҒҸгҖҒдәҲжё¬гҒ®зҜ„еӣІгҒ«йҒҺгҒҺгҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҖӮ

Kдёӯй–“еӯҗгӮ„Bдёӯй–“еӯҗгӮӮгҒҜгҒЈгҒҚгӮҠгҒ—гҒҰгҒҫгҒӣгӮ“гҒ§гҒ—гҒҹгҒ—гҖӮ

зҙ зІ’еӯҗзү©зҗҶгҖҒе®Үе®ҷзү©зҗҶгҒҜгҒҫгҒ гҒҫгҒ гҒ“гӮҢгҒӢгӮүзҷәеұ•гҒҷгӮӢеҸҜиғҪжҖ§гҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

1970е№ҙд»ЈгҒ®зҗҶи«–гҒҢе®ҹиЁјгҒ•гӮҢгҒҰгҒ„гӮӢж®өйҡҺгҒӢгҒӘгҖӮ

з§ҒгҒҢеӯҰз”ҹгҒ гҒЈгҒҹй ғгҒҜгҖҒзҗҶи«–зү©зҗҶеӯҰгҒ§гҒҜгӮ№гғҲгғӘгғігӮ°гӮ№зҗҶи«–гҒ§гҒ®гҖҢж¬Ўе…ғгҒ®гҒҹгҒҹгҒҝиҫјгҒҝгҖҚгҒЁгҒӢгҒҢжөҒиЎҢгҒЈгҒҰгҒҹгҒӢгҒӘгҖӮ

ж•°еӯҰзҡ„гҒӘзҗҶи«–гҒҜе…ҲиЎҢгҒ—гҒҰгҒҹгҒ‘гҒ©гҖҒиЁјжҳҺж–№жі•гҒҜгҒ•гҒЈгҒұгӮҠиҰӢгҒӨгҒӢгӮүгҒӘгҒ„жҷӮд»ЈгҒ гҒЈгҒҹгҒ‘гҒ©гҖҒгҒ„гҒҫгҒҜгҒ©гҒҶгҒӘгӮ“гҒ гӮҚгҒҶгҒӯгҖӮ

иӘһеӯҰгҒ«иә“гҒ„гҒҰгҖҒдҝ®еӯҰгҒҜгҒӮгҒҚгӮүгӮҒгҒҹгҒ‘гҒ©гҖҒз§ҒгҒ®иғҪеҠӣгҒҜзү©зҗҶеӯҰгҒ«еҗ‘гҒ„гҒҰгӮ“гҒ гӮҚгҒҶгҒӢгғ»гғ»гғ»гҖӮ

SACDгҒ§гҒӘгҒ„гҒӢгҒӘгҖҒгҒ§гҒӘгҒ„гӮҲгҒӘгғ»гғ»гғ»гҖӮ

зҸҫеңЁгҖҒеқӮжң¬зңҹз¶ҫгҒ•гӮ“гҒ®гҖҢeverywhereгҖҚгҒ®жҘҪжӣІгӮ’PCгҒ«еҸ–гӮҠиҫјгӮ“гҒ§гҖҒ192kHz/24bitгҒ«гӮўгғғгғ—гӮігғігғҗгғјгғҲдёӯгҖӮ

гӮігғігғ—гғ¬гғғгӮөгҒҢеҠ№гҒҚйҒҺгҒҺгҒҰгҒ„гҒҰгҒӣгҒЈгҒӢгҒҸгҒ®SHMCDгҒӘгҒ®гҒ«йҹігҒҢе®ҡдҪҚж„ҹгӮ„еҘҘиЎҢгҒҚж„ҹгҒҢгҒӘгҒҸгҒӘгҒЈгҒҰгҒ„гҒҫгҒҷгҖӮ

гӮўгғғгғ—гӮігғігғҗгғјгғҲгҒ®гғӯгӮёгғғгӮҜгҒ«гӮҲгҒЈгҒҰгҒҜгҖҒйҹійҮҸгғҗгғ©гғігӮ№гҒҢеҘҪгҒҫгҒ—гҒҸгҒӘгӮӢгҒ®гҒ§гҖҒгҒқгӮҢгӮ’жңҹеҫ…гҒ—гҒҰгӮігғігғҗгғјгғҲгҒ—гҒҰгҒҫгҒҷгҖӮ

гҒӣгҒЈгҒӢгҒҸиүҜгҒ„дҪңе“ҒгҒӘгҒ®гҒ§SACDгҒӘгҒ©еҮәгҒ—гҒҰгҒҸгӮҢгҒӘгҒ„гҒӢгҒӘгҖӮ

SACDгҒЁгҒӢиҒһгҒ„гҒҹгӮүгҖҒCDгӮ„гғҖгӮҰгғігғӯгғјгғүгҒӘгӮ“гҒҰйҒёжҠһиӮўгҒҜгҒӘгҒҸгҒӘгҒЈгҒЎгӮғгҒҶгӮ“гҒ гҒ‘гҒ©гҒӘгҖӮ

Play For Sure

з§ҒгҒ®жҢҒгҒЈгҒҰгҒ„гӮӢDENON AVC-4320гҒҜгҖҒWindows VISTA / 7гҒ§гҒ®Play For Sure гӮ’еҸ–еҫ—гҒ—гҒҰгҒ„гҒҫгҒҷгҖӮ

LANгӮ’йҖҡгҒ—гҒҰгҖҒгғӘгғўгғјгғҲеҶҚз”ҹгҖҒгғЎгғҮгӮЈгӮўгӮөгғјгғҗгғјгҒӢгӮүгҒ®еҶҚз”ҹгҖҒWebгғ–гғ©гӮҰгӮ¶гӮігғігғҲгғӯгғјгғ«гҒҢеҲ©з”ЁгҒ§гҒҚгҒҫгҒҷгҖӮ

гҒ“гӮҢгҒҫгҒ§гғЎгғҮгӮЈгӮўгӮөгғјгғҗгғјж©ҹиғҪгҖҒWebгғ–гғ©гӮҰгӮ¶гӮігғігғҲгғӯгғјгғ«гҒҜдҪҝгҒЈгҒҰгҒ„гҒҫгҒ—гҒҹгҒҢгҖҒгғӘгғўгғјгғҲеҶҚз”ҹгҒҜдҪҝгҒЈгҒҹгҒ“гҒЁгҒҢгҒӘгҒӢгҒЈгҒҹгҒ§гҒҷгҖӮ

Windows Media PlayerгҒ§еҶҚз”ҹеҸҜиғҪгҒӘгӮӘгғјгғҮгӮЈгӮӘгғ•гӮЎгӮӨгғ«гӮ’гҖҒWindowsдёҠгҒ§йҒёгӮ“гҒ§гҖҢгғӘгғўгғјгғҲеҶҚз”ҹгҖҚгӮ’гӮҜгғӘгғғгӮҜгҒҷгӮӢгҒЁгҖҒ

гҒ“гӮ“гҒӘз”»йқўгҒҢеҮәгҒҰAVгӮўгғігғ—гҒӢгӮүйҹігҒҢеҮәгҒҰгҒҚгҒҫгҒҷгҖӮ

гӮӘгғјгғҮгӮЈгӮӘгғ•гӮЎгӮӨгғ«гӮ’гӮ№гғҲгғӘгғјгғҹгғігӮ°гҒЁгҒ—гҒҰAVгӮўгғігғ—гҒ®йҖҒеҮәгҒ—гҖҒAVгӮўгғігғ—еҒҙгҒ§гғҮгӮігғјгғүгҒ—гҒҰеҶҚз”ҹгҒҷгӮӢгӮӮгҒ®гҒ§гҒҷгҖӮ

AVгӮўгғігғ—гҒ®жҖ§иғҪгҒ«й јгӮӢгҒ®гҒ§гҖҒгӮөгӮҰгғігғүгӮ«гғјгғүгҒҢгҒ—гӮҮгҒјгҒӢгӮҚгҒҶгҒЁй«ҳйҹіиіӘгҒ«гҒӘгӮҠгҒҫгҒҷгҖӮ

жңҖиҝ‘гҒ§гҒҜONKYOгҒ®AVгӮўгғігғ—гӮӮеҜҫеҝңгҒ—гҒҰгҒ„гӮӢгӮҲгҒҶгҒ§гҒҷгҖӮ

Where is KMixer.sys in Windows7?

гҒЎгҒҫгҒҹгҒ§гҒҜгҖҢгӮ«гғјгғҚгғ«гғҹгӮӯгӮөгғјгҒ§йҹігҒҢжӮӘгҒ„гҖҚгҒЁгҒ„гҒҶи©ұгҒҢжөҒгӮҢгҒҰгӮӢгӮ“гҒ§гҒҷгҒҢгҖҒгҒ“гӮҢгҒЈгҒҰKMixer.sysгҒЈгҒҰгҒ„гҒҶгӮӮгҒ®гҒ®гҒ“гҒЁгҒӘгӮ“гҒ§гҒҷгҒҢгҖҒWindows 7 / Windows VISTAгҒ«гҒҜгҖҢгҒӘгҒ„гҖҚгӮ“гҒ§гҒҷгӮҲгҒӯгҖӮ

гҒқгӮӮгҒқгӮӮSound / Audio device гҒёгҒ®гӮўгӮҜгӮ»гӮ№ж–№жі•гҒҢе®Ңе…ЁгҒ«иҰӢзӣҙгҒ•гӮҢгҒҰгҖҒXPд»ҘеүҚгҒ®ж§ӢйҖ гӮ’гҒ—гҒҰгҒ„гҒӘгҒ„гӮ“гҒ§гҒҷгҖӮ

XPд»ҘеүҚгҒ®з’°еўғгҒ§гҒҜгҒ“гҒ®KMixer.sysгӮ’еӣһйҒҝгҒҷгӮӢгҒҹгӮҒгҒ«ASI4ALLгӮ’д»ӢгҒ—гҒҰгҖҒdeviceгҒ«гӮўгӮҜгӮ»гӮ№гҒ—гҒҰгҒ„гҒҹгӮ“гҒ§гҒҷгҒҢгҖҒVISTA/7д»ҘйҷҚгҒ§гҒҜгҒ“гҒ®гӮўгӮҜгӮ»гӮ№гӮ’зҰҒжӯўгҒ—гҒҰгҖҒд»ЈгӮҸгӮҠгҒ«дҪҺгғ¬гӮӨгғҶгғігӮ·гҒ§гҒ®гӮўгӮҜгӮ»гӮ№гҒЁгҒ—гҒҰWASAPIгӮ’з”Ёж„ҸгҒ—гҒҹгӮҸгҒ‘гҒ§гҒҷгҖӮ

гӮўгғ—гғӘгӮұгғјгӮ·гғ§гғігҒӢгӮүгҒҜгҖҒWindows Core Audio API гӮ’д»ӢгҒ—гҒҰгғҮгғҗгӮӨгӮ№гҒ«гӮўгӮҜгӮ»гӮ№гҒ—гҒҫгҒҷгҖӮ

гҒ“гҒ®APIгҒ®дёӯгҒ«WASAPIгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

гҒ“гҒ®Core Audio API гҒҜгҖҒXPд»ҘеүҚгҒ«гҒҜжҗӯијүгҒ•гӮҢгӮӢгҒ“гҒЁгҒҜгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮ

е®ҹйҡӣгғ—гғӯгӮ°гғ©гғҹгғігӮ°гҒ®иҰӢең°гҒӢгӮүгҒ„гҒҶгҒЁгҖҒ

VISTA / 7гҒ§гӮөгӮҰгғігғүгӮ’йіҙгӮүгҒҷйҷҗгӮҠKMixerгӮ’дҪҝгҒҶгҒ“гҒЁгҒҜгҒӮгӮҠеҫ—гҒӘгҒ„гҒЁгҒ„гҒҶгҒ“гҒЁгҖӮ

WASAPIгӮ’дҪҝгҒЈгҒҹгғ—гғӯгӮ°гғ©гғҹгғігӮ°гӮ’гҒ—гҒҹгҒӢгӮүгҒЁгҒ„гҒЈгҒҰй«ҳйҹіиіӘгҒ«гҒӘгӮӢгҒЁгҒҜгҒ„гҒҲгҒӘгҒ„гҒ“гҒЁгҖӮ

гғүгғ©гӮӨгғҗгҒ®е“ҒиіӘгҒҢйҹіиіӘгҒ«дёҺгҒҲгӮӢеҪұйҹҝгҒҢеӨ§гҒҚгҒ„гҒЁгҒ„гҒҶгҒ“гҒЁгҖӮ

гӮӮгҒЈгҒЁгӮӮйҮҚиҰҒгҒӘгҒ®гҒҜгӮөгӮҰгғігғүгӮ«гғјгғүгҒ®е“ҒиіӘгҖӮ

44.1khz/16bit гҒӢгӮү192khz/24bitгҒ«гӮўгғғгғ—гӮігғігғҗгғјгғҲгҒҷгӮӢжҷӮгҒ®йҹіиіӘгҒҜгҖҒгҒ©гҒҶгҒ„гҒЈгҒҹгғӯгӮёгғғгӮҜгҒ«гӮҲгӮҠгҒ©гҒҶгҒ„гҒЈгҒҹдҝЎеҸ·иЈңй–“гҒҷгӮӢгҒӢгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҷгҖӮ

гғӘгӮўгғ«гӮҝгӮӨгғ гҒ«гӮҪгғ•гғҲгӮҰгӮ§гӮўгҒ§гӮўгғғгғ—гӮігғігғҗгғјгғҲпјҹ

гғӘгӮўгғ«гӮҝгӮӨгғ гҒ«гғҸгғјгғүгӮҰгӮ§гӮўгҒ§гӮўгғғгғ—гӮігғігғҗгғјгғҲпјҹ

дәӢеүҚгҒ«гӮҪгғ•гғҲгӮҰгӮ§гӮўгҒ§гӮўгғғгғ—гӮігғігғҗгғјгғҲпјҹ

гҒ©гӮҢгҒ§гӮӮгҒ„гҒ„гӮ“гҒ§гҒҷгӮҲгҖӮ

зөҗеұҖгҒ®гҒЁгҒ“гӮҚгӮўгғғгғ—гӮігғјгғігғҗгғјгҒЁгҒҷгӮӢгҒЁгҒҚгҒ®гғӯгӮёгғғгӮҜгҒҢйҮҚиҰҒгҒӘгӮ“гҒ§гҒҷгӮҲгҒӯгҖӮ

гӮӮгҒЁгҒ®е“ҒиіӘгӮ’и¶…гҒҲгҒҹй«ҳйҹіиіӘгҒҜгҒӮгӮҠеҫ—гҒӘгҒ„гҒ®гҒ§гҖҒгҒ©гӮҢгҒҸгӮүгҒ„гҒӮгӮҠеҫ—гҒӘгҒ„гғҮгғјгӮҝгӮ’дёҠжүӢгҒ«дҪңгӮӢгҒӢгҒҢгҖҒгғ—гғӯгӮ°гғ©гғһгғјгӮ„гӮігғігғҗгғјгӮҝгғјиЁӯиЁҲиҖ…гҒ®и…•гҒ®иҰӢгҒӣжүҖгҒӘгҒ®гҒ§гҒ—гӮҮгҒҶгҖӮ

гҒқгӮҢгҒҜгғ•гӮЈгғ«гӮҝгғјгҒЁе‘јгҒ°гӮҢгӮӢгӮӮгҒ®гҒ§гҒҷгҒҢгҖҒеҚҳзҙ”гҒ«гғҸгӮӨгғ“гғғгғҲгӮөгғігғ—гғӘгғігӮ°гҒ—гҒҹгҒ гҒ‘гҒ§гҒҜгӮ®гӮ¶гӮ®гӮ¶гҒҢгҒҹгҒЈгҒҹиҒһгҒҚиӢҰгҒ—гҒ„гғҮгғјгӮҝгҒ«гҒӘгӮӢгҒ®гҒ§гҖҒгғҺгӮӨгӮәгӮ’гҒӮгҒҲгҒҰд№—гҒЈгҒ‘гҒҰиҒһгҒҚгӮ„гҒҷгҒ„жіўеҪўгғҮгғјгӮҝгӮ’дҪңгӮӢгҒЁгҒ„гҒҶгҒ“гҒЁгҒ§гҒҷгҖӮ

гҖҢйҹіж•°гҒҢеў—гҒҲгӮӢгҖҚгҒЁгҒӢгҖҢжғ…е ұйҮҸгҒҢеў—гҒҲгӮӢгҖҚгҒЁгҒӢгҖҒзўәгҒӢгҒ«еў—гҒҲгӮӢгӮ“гҒ§гҒҷгҒҢгҖҒиҷҡж§ӢгҒ§гҒҷгҖӮ

иҒһгҒҸж–№гҒҜгҖҢй«ҳйҹіиіӘгҖҚгҒ§гҒҜгҒӘгҒҸгҖҒгҖҢеҘҪгҒҚгҖҚгҒӢгҖҢе«ҢгҒ„гҖҚгҒ§гҒ—гҒӢгҒӘгҒ„гҒЁгҒ„гҒҶгҒ“гҒЁгӮ’еүҚжҸҗгҒ«иҖғгҒҲгҒҹж–№гҒҢжӯЈгҒ—гҒ„гҒ§гҒҷгҖӮ
