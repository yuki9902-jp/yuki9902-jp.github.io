2010-04

どうしようっかな

今日はこれから明日にかけて冷えこむようです。

寒いです。

断熱材の入っていない家なので暖房切ると寒い・・・。

明日の日中も冷えこむようですが、例年並みらしい。

さて明日はまた桜と桃と躑躅の写真を撮りに出かけましょうかね。

今日の吉野山は残念な状態だったですが、リベンジは来年かな。

山の中に入ってしまうと桜に彩られた山全体が見えないから、絵になりにくいね。

それよりも奈良県内は川沿いにきれいに桜が植わっているのでそちらのほうがきれいだった。

今度奈良に行くときにはいらちの親といくんじゃなくて、一人か友達と行くべきかな。

いらちすぎてちょっとでも込んでると見て回れないから。

今日の吉野山

帰ってきた。

約６００キロを移動した。

今日は奈良県吉野山に行ったんですが、あいにくの雨。

PENTAX K-7 / smcPENTAX DA*50-135mmF2.8ED[IF]

明日は

明日はめっちゃ早いのでもうお休みなさい・・・。

今日の桜

「たけべの森」

PENTAX K-7 / smcPENTAX F*300F4.5EDD[IF]

「宇甘川」

PENTAX K-7 / smcPENTAX DA*50-135mmF2.8ED[IF]

春には体調不良

朝食をとったあと、写真を撮りに出る元気がなくて家にこもってました。

水曜日あたりに奈良の吉野に桜を撮りに出かけることに決めたので、すでにそれ以外の桜に興味をなくしてしまってます。

近場に１５０００本の桜を植えた「たけべの森」という公園があるのでこればかりは撮り頃になったらいってみようかな。

さて朝から元気がなくて、昼間ずっと寝てました。

実際、横になっているだけじゃなく寝入ってました。

夕食の時間になってようやく動き出したかな。

まあ今も元気がでないので、漫画「D.gray-man」を大人買いしたので読んでます。

なんか体のなかがよどんでいるようです。

どうしたものか

どうしたものかな。

県内の桜を撮るのが若干飽きてきた。

目に見て迫力があれど、写真にするとおもしろみがなくなるし・・・。

表現の問題か。

目

やっべ、まったく目のピントがあわない。

疲れて眼圧が下がっているんだろうな。

文字が全く見えないや。

昼寝しよ・・・。

児島繊維祭り

児島繊維祭りに行ってみようとしたけど、駐車場に入れず諦めました。

その後する事もなくウロウロ。

両親と一緒だから行く所限られるなぁ。

Cutting Edge

ただいま、NHK HIVISIONでCutting Edgeと言う番組が放送中です。

映画編集の話です。

とてもおもしろい。

自分でもやってみたい領域なのでめちゃくちゃ興味があります。

でもいまさらこの分野のプロにはなれないから、趣味の延長でしかないけどね。

FAUSHON

FAUSHONってパン屋もやってるのね。

コーポレートアイデンティティーのデザインがいつの間にか変更されていて現代風ではありますが、最初は気づかなかった。

今日、ダージリンティーとジャムを購入しました。

明日煎れて、そしてトーストを楽しもうかな。
