ONKYO SE-90PCI

ONKYO SE-90PCIのチップは、VIAのEnvy24 MTが採用されています。

このドライバーは、ONKYOはチップメーカーのものをそのまま使っています。

長らくWindows VISTA x64では、Microsoftから提供されるドライバーを使っていました。

途中、5月の始め頃に一度そのドライバーがアップデートされました。

そしてようやくチップメーカーからドライバーのリリースが行われました。

これで、Windows VISTA x64でも24bit-192kHz出力が可能になります。

また、SPDIFからDVDなどのDTSやDolbyDigitalのパススルー出力も可能になります。

早速、192kHzの出力をしようとしましたが、192kHz入力ができるデジタルアンプはありません。

96kHzは可能なので、96kHzに設定しました。

これに設定するとSPDIFのパススルー出力はできません。

192kHzはPIN端子のアナログ出力を使えば、かなりの高音質で出力できます。

でも、

１．24bit96kHzファイル→96kHz→光デジタル→デジタルアンプ

２．24bit96kHzファイル→192kHzDAC→VLSC回路→アナログPIN→デジタルアンプ

上記二つのどちらが良いかは、よくわかりません。

１．ではアンプに電気的に分離した光デジタル端子で入力できます。

それでも電源は、同じ所からとっているので、本当に分離しているわけではないですが。

２．では計測値S/N比110dbとなっていますが、PCの電磁波ノイズの影響をPIN端子やアナログケーブルで受けてしまっていないでしょうか。

ただし、PCノイズはMhzやGHz帯で22kHz以下の聴感域に入ってくるノイズではないので大丈夫だとは思っているのですが。

SE-90PCIの特徴はアナログ回路ですから、それを使わないとメリットが薄いかもしれません。

でも、デジタルアンプもシーラスロジックのDACとマランツの定評あるアナログ回路ですから、アンプ側でのD/A変換でも高音質です。

まあ、本当に高音質で聞く必要があるときは、すべての電気機器の電源を抜き、SACDプレーヤーからアナログ出力させアンプでダイレクト入力しアナログ回路だけで鳴らします。

シーラスロジックのDACとSONYの1bitDAC、どちらが高音質でしょうか。。。

Safari 3 パブリック Beta

Appleの日本語ページでも紹介されていた Safari 3 パブリック Beta 。

Safari は、シェアは決して高くはないですが、Webサイト作成では、それなりにデザインの考慮が必要になります。

自宅では、Windows Vista Ultimate x64 と Windows XP Professional SP2 の環境で、Internet Explorer 7、FireFox 2.0.4、Opera 9 は、確認作業ができますが、MacOSXは持っていないのでその環境でのテストは、できないでいました。

でも、Windows PC 版が公開されたことにより、それなりにシェアが拡大しそうです。

現時点の http://www.apple.com/jp/safari/で公開されているベータ版はセキュリティーホールやバグ、日本語が表示できないなどの問題があり、テストに使えるものではありませんが、アプライド版が出たあかつきには、FireFoxのシェアを喰ってしまいそうです。

現時点でInternet Explorer で妥協している人（他のブラウザを知らない人）が、Safari に乗り換えることはあまり期待できないですが、Anti Internet Explorer の人はすでにFireFoxを使っているでしょうから、そちらからSafariへの乗り換えが起こりそうな気がします。

業務でMacOSXでSafariを使うことがありますが、FireFoxより何が良いかと聞かれても、現時点では、速度面の優位性しか持ち合わせていないような気がします。

あのUIのデザインがベストだと言われても？？しかでないです。

Windowsよりはぼったりした感じがないですが、今となっては何となく陰気な感じがします。

あまり独自のUIを付けて欲しくないというのが正直なところです。

SNS構築日誌2

余ったPCをサーバーにしようと四苦八苦。

FedoraCore6をインストールしたものの、FTPサーバーもWWWサーバーもうまく建てられず、断念。

初期セットアップで、ApacheもPHPもMySQLもすべてインストールされるのですが、インターネットを参考にしていたのですが、そこからの設定がどうも簡単にはいかず、半日を費やしてしまいました。

いかんせん時間がないので自宅サーバーはちょっと諦めてレンタルスペース上にSNSをインストールすることにしました。

レンタルスペースのLolipopにOpenPNEをインストールしました。

MySQLもPHPも環境がすでに整っていますし、先人もたくさんいるのでそれらのサイトを参考にインストール。

今度は非常に簡単にインストールできました。

無事に稼働しています。

今、勉強会に使っているのでそちらで実績を積んで商品にしたいと思ってます。

OpenPNEは、話によると大手SNSで採用されているほぼそのままらしいです。

いわゆるMixiですが。

ほかにも大手のサイトが採用してるらしいです。

SNS構築日誌 1

WWW技術の勉強会で、SNSを構築してみようかと思います。

まずサーバーを準備しなくてはなりません。

サーバーは手元に転がっているスリムPCがあります。

CPUはAMD Sempron 3000+ (2GHz)で、メモリは1GB搭載しています。

テスト用WWWサーバーには十分なスペックです。

OSは、FedoraCore6にしようかと思います。

別にLinuxでApacheが動けばなんでもOKなんですが。

SNSは、オープンソースのOpenPNEなどが有名ですが、結構なカスタマイズが必要だとか。

でも、オープンソースこそ長いものには巻かれろ的な考え方でないと、参考資料が少なくて。

この土日あたりに動かしてみようかと思います。

Windows Vista Ultimate x64 (続)

すでにWindows Vista Ultimate x64 にしてから、１週間が経ちましたが、世間で言われるほどトラブルに遭遇してはいません。

ソフトウェアも大部分を切り捨てたので、別段、諦めさえあれば、それで快適に使えるものです。

サウンドカードは、ドライバーがメーカーからx64用がリリースされていないので、しばらく待たなければなりませんが、Microsoftから配布されるWDMドライバーで、とりあえずは動いています。

カードは、ONKYO SE-90PCI (VIA Envy24MT を搭載、24bit/192kHz対応)なのですが、16bit/48Khzまでの設定しかできません。

また、QSoundに対応しているようですが、それも享受できません。

しばらく待ちの状態です。

Windows XP Professional SP2では、Creative SoundBlaster Audigy2 NX (USB)を使っていたのですが、Windows Vista Ultimate x64 でインストールされるドライバーでは、激しく音割れします。

元来ＰＣ筐体の中は電磁波の嵐が吹き荒れているので、サウンドカードは外付けにしたいのですが、とりあえず音を出すためにONKYO SE-90PCIを搭載しました。

マザーボードにもサウンドチップが搭載されていますが、AC'97なのでHDSoundには対応していないし、16bit/48kHzまでの設定までしかなかったので。

あと、オーディオプレーヤー関係は今のところ全滅ですかね。

まあ、HDD Walkmanを使っているので、Playstation3上で利用できるので、あまりこちらで困ることはないかな。

でも、取り溜めていたATRAKファイルは一度捨てないとだめかな。

あと、Adobe関係のソフトウェアはきちんと購入して正規のライセンスを取得しようかと。

まあ、それが実現できる頃には、Adobe Creative Suite 3 もx64に対応していることでしょう。

x64でも32bitのソフトウェアしかないのなら宝の持ち腐れだから。

ああ、一番問題なのは、Microsoftのサイトかな。

Vistaに付属のソフトウェアに関するページがなかったり、英語のままだったり、不備が多い。

付属しているソフトウェアにはきちんと責任を持ちましょう。

ユーザーはそれにもお金を払っているのだから。

使えないままなら、お金を返すぐらいの覚悟が必要だよ。一応企業なんだから。

Color Management

Windows Vista から、色管理が簡単になるとのことで、少し調べてみましたが、あまり情報が公開されておらず、概論的なことだけしかつかめません。

色管理はコントロールパネルにあるのですが、どう設定すべきなのかヘルプもなくわかりません。

手持ちの液晶ディスプレイには、ICCプロファイルがついているのでとりあえずインストールしておきました。

ディスプレイの色温度設定は、sRGBにしています。

一応、1677万色表示が可能なパネルではありますが、sRGBの範囲内の1677万色でしょうかね。

また、黒の再現性に乏しいので、厳密な色あわせには使えません。

まあ、それほど厳密さを求めるほどのことに使ってないのでいいのですが。

日本工業規格(JIS)での色調整は、5500Kの標準光下で行わなければならないそうですが、じゃあ、ディスプレイも5500Kにしなくてはいけないのかな。

sRGBに設定したディスプレイは、7200Kよりも若干黄色っぽく、6500Kよりも若干青っぽいです。

室内の蛍光灯は、TOSHIBA ネオスリムZ 3波長タイプ昼光色なので6700Kです。

と、いうことは印刷物を確認するのは、6500K～6700Kのディスプレイ設定と環境光で確認する必要があります。

ちょっと前まで、その調整をしていたのですが、Vistaでは、どの程度調整したらいいのでしょうか。

この辺、どっかで詳しく解説したサイトがないかな。

Windows Vista Ultimate x64

Windows Vista Ultimate x64をインストールしました。

今までの環境をずばっと切り捨てました。

x64環境では、ハードウェアのドライバ周りがそろっていないので、いろいろと制限がありますが、そこは買い換えの理由にしてしまおうと思っています。

アプリケーションもハードウェアにくっついてきたものが多く、そのほとんどを使っていません。

動画編集やDVDエンコーダ、簡単写真加工ソフトなど、全く使っていません。

使っているのはごく限られたソフトだけ。

Photoshop7、秀丸、NextFTP、DreamWeaver、Microsoft Office 2003、一太郎2006、ATOK2006、Shuriken Pro 4、Sony SonicStage CP、Picasa2、B’s Recorder GOLD7ぐらいですかね。

このうち、Vista x64に対応していないのが、SonicStage CP、B’s Recorder GOLD7ですか。

まだ、NextFTPを試していません。

まあ、おいおい対応したソフトが出てくるでしょう。

そろそろPhotoshopCS3が登場しそうなので、本腰入れて、Web開発環境を整えようかな。

さて本題に戻って、Vista x64のインストールは簡単。

特に今回は、RAIDを構成していないのであっさりインストールできました。

難点は音源にCREATIVE SoundBlaster Audigy2 NXを使っていましたが、ドライバーがインストールされ、何となく音が出るのですが、完全に割れてしまっています。

これまで、サブ機に搭載していたONKYO SE-90PCIを取り付けて、ドライバーを自動インストール。

出力のbit/周波数に制限があるものの、音割れはありません。

まあこれもしばらくすればドライバーが提供されるでしょう。

グランツーリスモHDコンセプトの無料ダウンロードのポスターも目に入り、売り場の前を何度も往復した末、決心。

ナビ取り付け後日誌 １

PanasonicのHDDナビCN-HDS620DをIMPREZA WRX STI (GH-GDBF)を取り付けて、使い勝手にそれなりに満足はしているものの、それはあくまで昼間での使用に関して。

夜間は、ボタンの青い照明が眩しくて、ちょっと左側が見にくくなりました。

画面はOFFにできるのでいいのですが、ボタンの照明はOFFにできません。

ナビ本体の電源を落とすこともできないので、ちょっと対策を検討しなければ。

夜の山道を攻めるときは、青い光が眩しく完全に邪魔になってしまいます。

ばらして、輝度の低いLEDに交換するか、ボタンの上に簡単なカバーをつけるか。

映り込みも気になるので、ボタンの上にカバーをつけるのが良いのかな…。

青は眩しいのでオレンジ系統に交換するのも良いかも。

Windows XP Professional x64

ようやくWindows XP Professional x64を正常起動できるようになりました。

原因は、ビデオカードのドライバーだったようです。

セーフモードで起動した後、最新のドライバーをインストールして再起動すると、何事もなかったかのようにきちんと起動しました。

SLI設定をしているのが問題だったのかもしれません。

[OS] Windows XP Professional x64 Edition
[CPU] AMD Athlon 64 x2 3800+
[MEM] 1GB * 2(Dual Channel)
[VGA] GeForce6600GT * 2(SLI)
[HDD] 60GB(ATA133)/160GB(ATA133toSATA) * 3
