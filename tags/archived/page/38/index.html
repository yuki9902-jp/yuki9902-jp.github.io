ODYSSEY - Halery Westenra

ヘイリーは、いわゆる声楽のジャンルになります。

透明感のある美声と柔らかな曲調で癒されます。

最近の疲れ気味の私には心地よいです。

Final Fantasy XII の 2

FFXIIのプレイ時間が、トータルで15時間ほどになりました。

「王墓」に入る直前って所ですか。

一度入って全滅しましたけど。

現在のマップでは適当にアイテムが転がっているので、レベル上げと一緒にアイテム探しでもしますわ。

まだまだ先は長い。

あなぶき興産のCM

あなぶき興産のCMに、山中千尋さんが出演されています。

始めてCMを見たときは、音楽スクールのCMかと思ってしまいました。

山中さんは、相変わらずアスリートのような筋肉美ですね。

あれだけパワーのある演奏なら、それもうなずけるってもんです。

あなぶき興産のCMなので、中四国、関西がメインかも知れませんが、目にすることがあれば。

Final Fantasy XII

FFXII 買いましたよ。

まだやってないですが。

昼のお弁当を買いにコンビニに行ったら陳列棚にあったので、「FF12ください」って言って購入しました。

なんだ、予約しなくても買えるじゃん。

そうそう、ポーションのプレミアムボックスも３つ購入しました。

もう少し、キャップの作り込みをして欲しかったです。

開栓後に奇麗にキャップが閉まるようにして欲しかった。

ちょっと世の中の流れに乗ってみました。

Windows XP Professional x64

ようやくWindows XP Professional x64を正常起動できるようになりました。

原因は、ビデオカードのドライバーだったようです。

セーフモードで起動した後、最新のドライバーをインストールして再起動すると、何事もなかったかのようにきちんと起動しました。

SLI設定をしているのが問題だったのかもしれません。

環境：
[OS] Windows XP Professional x64 Edition
[CPU] AMD Athlon 64 x2 3800+
[MEM] 1GB * 2(Dual Channel)
[VGA] GeForce6600GT * 2(SLI)
[HDD] 60GB(ATA133)/160GB(ATA133toSATA) * 3

Picasa

googleが提供する技術というものは、新しいし、非常におもしろいし、意欲的ですごいなと常に感心します。

最近、画像ビューアーとして、よく使うようになったのが「Picasa2」です。

サムネイルの表示速度、スクロールのなめらかさ、画質調整など優秀で使い勝手もなかなか良いです。

しかし、画像検索時の安定度に若干問題があるのと、スライドショーの機能が少し弱いことが惜しいところです。

このツールの今後に期待です。

POTENZA RE050

スタッドレスタイヤからラジアルに交換しました。

昨シーズンは、POTENZA RE-01R だったのでちょっと変化を求めて、今シーズンはPOTENZA RE050にしました。

インアウトの非対称パターンで、ローテーションはありません。

排水性を重視したパターンで、ウェットコンディションでも良好なグリップを得られるのが特徴です。

履き替え後にすぐに雨天となり、ウェットでのグリップ性能を確認することができました。

比較はRE-01R、RE-01となります。

目的の違いが明らかなので、良い比較ではありません。

明らかにウェットでのグリップはRE050が良好です。

あらゆるラジアルタイヤの中でもトップクラスかも知れません。

ドライでのグリップは、RE050でも悪くありません。

むしろハイグリップタイヤと言っても大丈夫でしょう。

ただ、転がり抵抗は非常に大きいです。

アクセルをゆるめるとみるみる減速します。

燃費が気になるところです。

RE-01Rは、転がり抵抗が小さく、燃費が意外に良かったのです。

サーキットで周回を重ねてもタイムが落ちないと謳っているのは伊達ではなく、グリップ向上とと転がり抵抗低下の効果に裏付けされているからです。

RE050は、ドライもウェットもいけるバランスの良いタイヤかも知れません。

如何せん、高いのとロード(パターン/タイヤ)ノイズが大きいのが気になりますが。

100ポイント回復

FINAL FANTASYXII POTION って、えぐい商売ですね。

周りの人が買ってきて、私ももらいました。

これで100ポイント回復です。

–

数量限定プレミアムボックスがあります。

すごく瓶が奇麗なので欲しいです。

二つで良いですか？

CPUをデュアルコアにして、メモリをデュアルバンド、HDDをミラーにしたとなると残るデュアルと言えば、やはりビデオカードでしょう。

そう、やりました。

ビデオカードの２枚差し。

かつてはVoodooというビデオカードがあり、バンシーを出した頃で終わったのですが、２枚差しで速度アップを実現していました。

何となくそれに似た要領でビデオカードを２枚差しにして速度アップを図るのがSLIです。

さすがにSLIは専用のチップセットを搭載していないと無理なのですが、SLIをすることを念頭に置いていたのでSLI対応マザーを購入していました。

CPU、マザー、１枚目のビデオカード、メモリを買った店にGOです。

同じビデオカードをGET。

ファンレスのGeForce6600GTです。

インストール作業は、SLIマザーのPCIExpressソケットにあるSLI設定基盤をひっくり返して差し直し、２枚目のビデオカードをセットするだけ。

OSを起動してベンチマークを起動。

おお、確かに２倍ぐらいの数値が出ています。

AOE3のグラフィックフルオプションでも少しぎこちないですがゲームができます。

さすがに最上位のビデオカードには遠く及びませんが、それでも十分に速いです。

CPU、マザー、メモリ、ビデオカードで１０万円ほどの投資が必要でしたが、今回のチューンアップはかなりの満足が得られました。

久々にPCゲームをしたくなりました。

Windows XP Professional x64

64bitCPUを購入したのだからと、勢いに乗ってWindows XP Professional x64を購入しました。

ユーザーインターフェイスとかその辺りは、Windows XP Professionalとさほど違いがありませんが、OS内部は全く別物です。

さて、インストールは無事に終了したものの、起動しません。

Windows XPの起動ロゴが表示されたあと、左上でカーソルが点滅する画面で、ドライバのロードが開始されますが、USB周辺をロードしたときに、再起動が発生してしまいます。

何度、チャレンジしても同じところで再起動です。

セーフモードで起動するので、マザーボードに搭載されたチップとかはすべて添付の64bit用ドライバをインストールしてやります。

しかし、解決できません。

自作の鉄則、周辺機器を外して起動すればいいのですが、面倒くさい。

途中で飽きて、デュアルブートにしたWindows XP Professionalを起動して利用しています。

まだしばらくは64bit環境に移行しないですね。

ドライバもそうですが、一般ソフトウェアが64bit用に提供されていないので、32bitエミュレータ動作となり、メリットがないです。

ユーザーを引きつけるようなインターフェイスや独自の機能が盛り込まれていないので、なかなか移行を促すようにはならないのでしょう。

しかし、Windows Vistaを見れば開発環境として提供されている考えるのが妥当であり、Vistaのリリースにはドライバやソフトが充実しているものと思ってます。（期待しています。）

もう少しすれば64bit環境が自宅でも利用できます。

すごい時代です。
