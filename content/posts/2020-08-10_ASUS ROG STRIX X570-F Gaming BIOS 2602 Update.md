[ASUS ROG STRIX X570-F GAMING BIOS 2602](https://www.asus.com/jp/Motherboards/ROG-Strix-X570-F-Gaming/HelpDesk_Download/)

BIOSが更新されました。

----------------------------------

ROG STRIX X570-F GAMING BIOS 2602

"Improve system performance and stability

Improve Fan control function

Improve DRAM stability

Update AM4 AGESA combo V2 PI 1.0.8.0

Improve system stability

Improve DRAM performance

Before running the USB flashback tool，please rename the BIOS file (SX570FG.CAP) using BIOSRenamer"

----------------------------------

更新内容は、

システムの性能と安定性の改善、

ファンコントロール項目の改善、

メモリー安定性の改善、

AM4 AGESA combo V2 PI 1.0.8.0の更新、

システムの安定性の改善、

メモリー性能の改善

USBフラッシュバックツールを使う前に、BIOSRenamerをつかってBIOSファイル（SX570FG.CAP)にリネームしてください。

ということです。

注意：8/13現在、このバージョンのBIOSは取り下げられています。

詳細は分かりません。

少し前にメモリーの動作が怪しくなっていたので、今回の更新で安定するようになれば良いですが。

またしばらく様子見です。

今のところプレシジョンブーストは、Extremeで電力供給を130%で設定していますが、私が持っているRYZEN 7 3700Xでは、全コア4.0GHz、シングル4.25GHzが限界のようです。

正直、前のGIGABYTE AX370-GAMING K7の方がRYZEN 7 3700Xの性能を引き出せていました。

元々X570は見送る予定だったんですが、X370ではDDR4 3600MHzの4枚差しが安定しなかったので、急遽購入したんです・・・。

GIGABYTEの製品は昔は安かったんですが、X570以降は代理店がかなり強気の価格設定をしているので、コストパフォーマンスが悪く手を出しづらいです。
