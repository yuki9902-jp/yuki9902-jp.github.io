Merry Christmas

今晩は。

良いイブを過ごせましたか。

明日の日没までが降誕祭です。

明日も仕事なので飲むわけにもいきませんが、ぼーっとして、なんとなく幸せな気分になってみます。

冗談です。

Sony VAIO P (VGN-P70H/R) Windows7 Upgrade

いったんWindows7 Home PremiumへUpgradeしたら、とても快適でした。

12/23

で、Sony から VAIO Windows7 サプリメントディスクが届いたので、サポートされる手順に従ってUpgradeし直します。 まず、リカバリーしてWindows VISTA Home Basic 32bit に戻します。 続いて、サプリメントディスクを入れて、事前セットアップ。 不要なアプリをアンインストールしろと書いてあるので、削除。 Windows 7のDVDを入れてアップグレード開始。 途中でエラーになって止まった。

ここに来て面倒なので、再起動してDVDブートでアップグレード開始。 どうせならクリーンインストールしようと、マニュアル見ると出来ると書いてある。それにしたがってクリーンインストール。 リカバリーディスクは作ったので、リカバリー用領域を削除して、パーティションを再設定。 Windows7のインストール開始。

とここまでを今朝の出勤の段階までやってきた。家に帰ったら出来ているかな。

12/24

先達のブログなどを見ると、リカバリー領域がないとVAIOアプリやドライバがインストールできないらしい。

またSonyのサイトには、クリーンインストールでは、アプリのインストールが出来ないものがあるらしいとのこと。

そんなん嫌ジャン。なので家に帰ったら、またリカバリーからやり直そうっと。

12/25

大多数の人はChristmasを満喫しているだろうが、明日も仕事。

お酒を飲むわけには行かない。

DVD-Rで作ったリカバリーディスクから、リカバリーを開始。

パーティションなど購入時の状態に復元。

リカバリー領域にデータがコピーされ、リカバリーが走り始めた。

あとは眠いので朝までほったらかし。

朝起きたらリカバリーが終了してた。

出勤する前にサプリメントディスクを入れて事前セットアップをして、ほったらかし。

12/26

毎日午前様なのがつらいところ。

Windows7では動かないアプリをアンインストールして、ついでに使わないアプリもアンインストールして、再起動。

Windows7のDVDを入れて、DVDブートしてセットアップしたのでは、アップグレードできない。

Windows VISTAを起動して、Windows7のDVDを入れて、Setup.exeを起動させる。

Windows7のセットアップのもろもろの工程を進めて、あとはほったらかし。

眠いので寝る・・・AM3:00

AM6:30出勤する前に、Windows7のプロダクトIDを入力して、アップグレード完了。

つぎにSonyのサプリメントディスクをインストール。

出発時間になったのでインストールをほったらかし。

ハヤテのごとく!! 第５巻

ハヤテのごとく!! 第５巻がリリースされました。

animateで夏に注文していたのが届きました。

「桂ヒナギク」萌えな私には、とても楽しみな巻です。

本放送ではうるっときちゃいました。見直しても少しじんわりと来ました。

そもそもここにブログ開いたのも、このアニメが切っ掛けで、伊藤静さんを追っかけてのことだしね。

いやはや影響がでかい。

Hayate no gotoku No.5 Blu-ray Limited

ハヤテのごとく!! の第５巻が届きました。

「桂ヒナギク」萌えな私としては、非常に楽しみな巻です。

ほぼ原作通りなので余計なカットもありますが、前半を締めくくる良い仕上がりです。「Feeling of freedom」放送時は、なぜか、そうなぜかうるっと来てしまったんです。

いやいや心が癒される良い巻でした。

どんな感じ

どんな休日を過ごしているかな

晩飯を食べにでると言われ両親についてきたのはしょうがないが、暇だね。

買い物でもできれば気は紛れるかも知れませんけど。

午前２時

02:00

JazzのNetRadioを聞きながら、Otard XOを口に含む。

少し小説を読んでみる。

冬は曙

氷点下５度、この冬１番の寒さです。

霜が降りて路面がうっすらと凍っていたようですが、太陽が出るとすぐに解けました。

室内との温度差、朝晩の冷え込みで体調を崩さないようにしなくては。

乾燥して咳が出るし。

でも明け方の空の色がきれいですよ。

なんとなく

なんとなく間が抜けてしまいました。

会社の経営方針の話をすることになるのかと思っていたのですが、今日は忘れてたのか、それがなかった。

おかげで間が抜けてしまいました。

ただ問題は先送りになったのだと思うので、気が気ではありません。

モラルハザードを起こしている会社ですから今更更正させようとするのは大変です。

無駄な労力だ。

私の時間を生きたいのにね。

SONY VAIO type P

中古で Sony VAIO type Pを買いました。

今年の１月に登場したモデルで、CPU Atom 1.33GHz / 2GB Ram / 60GB HDD / ワンセグモデル というスペックです。

カラーはガーネットレッド。

価格は、46800円。

OSは、VISTA Home Basicなので、即効Windows7 Home Premiumを通販で購入しました。

使用感はいたるところに出ているように、HDDがボトルネックになっていて遅いです。

Windows7をインストールしてどれほど変わるか試してから、SSDを購入するか検討します。

某社のSSDと交換しても、さほど速度アップが見込めないことから、どうせ交換するなら、1.8インチのHDD(ZIF)の大容量を買ったほうが安くていいかもしれませんね。

でも1.8インチのHDDのほうが希少かも。

SONY VAIO P

中古のSony VAIO Pを購入しました。Atom 1.33Ghz 2GBRAM HDD60GBという仕様です。

Windows VISTA Home Basic だったので、さくっとWindows 7 Home Premium Upgradeを注文しました。

動作はHDDが遅いのがかなりのネックですね。SSDに交換すれば早くなるかな。VISTAの起動はかなり時間が掛かりますが、ハイバネーションを使うのであまり気にならないかな。

使いにくいのはアキューポイントだからですが、慣れたとしても使いづらい。でも小さいことのメリットは非常に大きいです。

Windows UpdateやらVAIOアプリのアップデートにすごい時間が掛かってます。まだおわんない。
