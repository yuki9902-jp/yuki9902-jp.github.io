自分の体温すら暑苦しい

暑さとしんどさに耐えかねて、ベッドに横になってますが、室温が30度あるし、ベッドのマットレスから反射してくる自分の体温すら暑苦しいです。

扇風機がないと寝られないし、朝起きると脱水してて、足がつるし、毎日キツいです。

エアコンはコンプレッサーのゴゴゴって音が家全体に響くし、梅雨寒からの急激な猛暑は体に堪えます。

でもねなくちゃ。

暑いでゲス

暑すぎて、日陰で扇風機を使おうと、熱中症になりました。

頭痛がひどいので、痛み止めを飲みましたが、晩ご飯を作っている間は耐えられましたが、それ以降はまた痛みがでてきました。

この時間になって30度をきったようです。

晴れていても湿気が多くて蒸し暑いです。

スマホ復活

Sony Ericsson XPERIA X Performance SO-04Hのバッテリー交換をして動かなくなっていた件、裏蓋を押し込んだらいよいよ何も反応しなくなったので、再度バラしてみました。

バッテリーと本体回路へのコネクター部分が少し斜めになって浮いていました。

それをしっかりと押し込んで、USBケーブルを挿すと、今度はスマホの充電状態を示す画面が表示されました。

ということで、起動しなかった原因はバッテリーの接続不良でした。

裏蓋に2ミリ幅の両面テープをきっちりと貼り、裏蓋をはめ込み、ちゃんと使えるようになりました。

昨晩に注文したXPERIA 8のキャンセルは出来ないので、新しいスマホが届き次第、データをそちらに移し、XPERIA X Perfomanceは親に譲ります。

2022年にはauのガラケーは使えなくなるので、それに対応すると言うことで、納得しよう。

スマホのバッテリーを交換したものの

スマホのバッテリーが届いたので交換しました。

しかし、起動したと思ったらすぐに液晶がオフになって、LEDが赤く3度点滅して、しばらくしてまた3度点滅して、を繰り返します。

強制電源オフやら、コネクタ清掃とかしましたが、回復しません。

情報がなかなか出てきませんでしたが、なんとか調べがつきました。

スマホの本体の3.3V系に異常が出ているときのLED表示で、本体の修理をしないと直らないようです。

OCNのSIM契約で購入した中古品なので、修理をするのも馬鹿らしいので、新古品で未使用品のXPERIA 8の購入手続きをしました。

しばらくLINEとEdyが使えませんが仕方ないです。

というかLINEやめたい。

ぱんぱん

今朝、スマホを見るとバッテリー残量が0になっていました。

マイクロUSB Bの本体側の接触が悪くなっていました。

端子にエレクトリックパーツクリーナーを吹きかけて、エアダスターで吹き飛ばしました。

そうするとちゃんと充電されるようになりました。

で、なんだか本体の裏側が一部浮いているのが分かりました。

バッテリーの寿命が尽きて膨れているようです。

ドライヤーで裏側を温めて、両面テープの粘着力を緩めて、カバーを剥がしました。

バッテリーがぱんぱんに膨れていました。

バッテリーも両面テープでがっちりくっついているので、それを剥がしてバッテリーを外しました。

で、密林でChina製の横流しバッテリーを注文しました。

明日には届くでしょう。

届いたら交換します。

ASUS ROG STRIX X570-F Gaming BIOS 2602 Update

ASUS ROG STRIX X570-F GAMING BIOS 2602

BIOSが更新されました。

ROG STRIX X570-F GAMING BIOS 2602

“Improve system performance and stability

Improve Fan control function

Improve DRAM stability

Update AM4 AGESA combo V2 PI 1.0.8.0

Improve system stability

Improve DRAM performance

Before running the USB flashback tool，please rename the BIOS file (SX570FG.CAP) using BIOSRenamer”

更新内容は、

システムの性能と安定性の改善、

ファンコントロール項目の改善、

メモリー安定性の改善、

AM4 AGESA combo V2 PI 1.0.8.0の更新、

システムの安定性の改善、

メモリー性能の改善

USBフラッシュバックツールを使う前に、BIOSRenamerをつかってBIOSファイル（SX570FG.CAP)にリネームしてください。

ということです。

注意：8/13現在、このバージョンのBIOSは取り下げられています。

詳細は分かりません。

少し前にメモリーの動作が怪しくなっていたので、今回の更新で安定するようになれば良いですが。

またしばらく様子見です。

今日も暑いです

長崎に投下された原爆で犠牲になった方々に哀悼の気持ちを捧げます。

今日も暑い日でした。

ここしばらく山の上は霞が掛かった日が続いています。

湿気が抜けずに非常に蒸し暑いです。

食料品の買い出しに行き、駐車場での外気温が46度を指してました。

ヤバいです。

車を走らせても36度でした。

山の上に帰ってくると31度でしたが。

さて、今の室温も31度です。

昼間は風が吹いていたんですが、夜になって風が止み、蒸し暑いです。

明日は4時起きで、大型ラジコンヘリで田圃に防虫剤を散布します。

これをしないと、カメムシが稲穂に集って、米が黒色に変色して、味が悪くなります。

無農薬なんて机上の空論ですよ、町の人は米を作る難しさを知らないんです。

鶏糞や牛糞を撒くと窒素量が多くなって、米がまずくなります。

適切な窒素、リン酸、カリウムの量と溶け出すタイミング、水の供給で稲の生育を調整して、安定した品質で食味の良い米が出来ます。

自然が為すがままに放置した田圃は不揃いの稲に、初穂も実入りもまばらで、収穫量が3分の1以下になります。

農業も時代に合わせて進化しているんですよ。

おなかがすいた

晩ご飯を早い時間に食べて、ほとんど汁物だったので、おなかがすいてきました。

そしておなかがくるくる鳴ってます。

最近は、昼からずっと麦茶を飲んでます。

朝はコーヒーを飲むことが多いですが。

麦茶は六条大麦、麦粒は潰していないもの、これが甘くて香ばしくて美味しいです。

HDDのサルベージ作業終了

HDDから必要なデータをサルベージして、新しく購入した不安定なSATA-USBブリッジを外して、もともと使っていたものに戻してHDDを初期化。

HDDの構成とデータを整理して、SSDのEFIパーティションのサイズがバラバラだったのを整理しました。

ようやく通常の運用環境に戻りました。

構築作業をやっているときは、時間を忘れてしまいます。

元の仕事がSIやSEだけにやっぱり向いているんだろうね。
