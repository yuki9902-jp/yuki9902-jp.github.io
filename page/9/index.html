二台目のASUS TUF Gaming VG27AQL1A

先月の18日に注文していた27インチWQHDディスプレイ ASUS TUF Gaming VG27AQL1Aが今月の15日に届きました。

壊れた37インチのFullHD TVは処分したので、空いたスペースにTVの棚の下に設置していたセンタースピーカーを棚の上に設置し、その上に27インチのWQHDディスプレイを設置しました。

一台目の27インチディスプレイと同じ型番の商品なので横位置の横並びにしても段差も色差もなく違和感が全くありません。

横並びで二画面を1つに纏めた設定で原神をプレイして見ましたが、ど真ん中でベゼルを挟んで左右に分かれ、キャクターもレティクルも見づらいので、横並びの配置は止めました。

メインを横位置で中央配置、サブを縦位置で右配置にして並べると非常に使いやすいです。

特にゲーム配信中の配信アプリ、コーディングの資料表示、ゲーム時の攻略情報表示、写真編集時の縦表示など非常に使いやすいです。

システムエンジニアだった頃、15年ぐらい前から仕事で19インチと17インチの5:4(1280*1024)を二台並べて使っていました。

メインが小さくなればサブはあった方が何をするにしても効率が良くなります。

今回はWQHD、170Hz、sRGB比130%の広色域、HDR対応と性能に拘ったため、結構高いですが、出来ることが多くなって良かったです。

稲葉浩志 - 羽 feat. DURAN

THE FIRST TAKEに稲葉浩志さんが登場するとは思っていませんでした。

その時々の声の違いを感じつつ、やはり肌が粟立つような感動を覚えました。

是非聴いてみて下さい。

起動ディスクをWD SN750からWD SN850Xに入れ替えました。

Amazonの期限限定ポイントを消費するために、WesternDigitalのWD BLACK SN850X 1TB（ヒートシンクなし）(NVMe Gen.4x4)を購入しました。

Amazonギフトカード、Amazonポイント、提携クレジットカードのポイントを駆使して約１万円分を引いて、7千円ほどで買いました。

NVMe-USBブリッジタイプの外付けSSDケースで、新しく購入したSN850Xを接続し、これまで使っていたWesternDigitalのSN750から、Acronis True Image For WDを使って起動ディスク用に複製します。

複製後は、PCをシャットダウンして、マザーボード上のSN750を取り外し、SN850Xに差し替えて、起動するだけです。

元が500GB、新しい物が1TBですが、Acronis True Imageで複製すると、空き領域を自動で拡張してくれているようです。

起動後は、Adobe製品などの認証も再度行われることもなくすぐに使用できました。

SN850Xのベンチマーク

SN750のベンチマーク（別の2TBの製品）

ベンチマークの結果はほぼ公称値通りで、シーケンシャルの読み書きが二倍以上になっていますが、起動時ではその違いをほとんど感じないです。

ただ大規模なアプリケーションの起動やインストールなどはかなり早くなっているように感じます。

WD DashboardでSSDの状態の確認やファームウェアの更新が出来るので、インストールしておくと良いと思います。

Elgato Wave Linkの音声遅延対策：USB DAC接続時のレイテンシを最小化する

Elgato Wave Linkの音声遅延対策

USB DAC接続時のレイテンシを最小化する

Elgatoの「Wave:3」や「Wave XLR」を使用する際、専用ミキシングソフトウェアである Wave Link のインストールが必須となります。しかし、接続構成によっては音声に無視できない遅延（レイテンシ）が発生することがあります。

1. 接続経路による遅延の比較

検証の結果、音声を出力するデバイスによって遅延の挙動が異なることが分かりました。

  • パターンA：外付けUSB DACを使用（遅延あり） [Wave XLR] ⇒ [PC(Wave Link)] ⇒ [仮想ドライバー] ⇒ [USB接続] ⇒ [USB DAC] この経路では、USBオーディオドライバの規格上、10ms〜30ms程度の遅延が発生します。
  • パターンB：PC内蔵DACを使用（遅延なし） [Wave XLR] ⇒ [PC(Wave Link)] ⇒ [仮想ドライバー] ⇒ [マザーボード内蔵出力] OS内部での処理で完結するため、数ミリ秒程度の遅延が発生しますがエコーを感じる程度です。
  • パターンC：OBSでの録画・配信（遅延なし） [Wave Link] ⇒ [OBS] OBSへの入力（Stream出力）に関しては映像との同期が取れており、録画・配信データにズレが生じることはありません。

2. USB DAC接続時の遅延を低減する方法

USB DACを使用してモニターする際の遅延を小さくするには、ドライバのバッファサイズを極限まで絞り込む必要があります。

Wave Link自体はWASAPI排他モードやASIOには対応していませんが、設定から入出力バッファを最小に設定することが可能です。

さらに踏み込んだ対策として、ツール「HIDUSBF」を使用する方法があります。これでUSBポートのポーリングレートを上げ、信号遅延を1msに固定することで、若干のエコーを感じる程度（PCのヘッドフォン端子直結と同等レベル）まで遅延を抑え込むことができました。

3. 配信・録画時の設定について

Wave Linkの「Stream出力」は、自分の耳で聞く「Monitor出力」よりも遅延が少なく設計されているようです。そのため、配信視聴者側で音声と映像がズレる心配はほとんどありません。

もし、それでもミリ秒単位のズレが気になる場合は、OBS側の「オーディオの詳細プロパティ」から同期オフセットを調整して、設定を追い込んでみてください。

痒い

パイナップルでアレルギーがでたようです。

顔に発疹して痒いです。

寒さと仕事疲れ、しばらく続いているリンパ腺の腫れ、いろいろあって、そこにアレルギーが出て、辛いです。

ステロイド系の肌荒れ用の薬を塗って、アレルギーが落ち着くのを待ちます。

食べられる果物が徐々に少なくなっていきます･･･。

今朝は雨

今日は未明から雨が降っています。

雨が続いているため、田圃がぬかるんで田起こしが出来ません。

田植えには間がありますが、ちょっと困りものです。

今朝も身体があちこち痛いですが、気分をすっきりさせるため紅茶を飲みます。

インドダージリン地方のナムリン農園のアッパー（上の方の茶畑）の2019年のFTGFOP1グレードの茶葉を煎れます。

軽やかでフルーティーな感じですが、黄砂で鼻の奥がやられてて、馬鹿舌になっててよく分からなくなってます。

はぁー。

15年使ったメインモニターが壊れました

2008年4月に発売されたパナソニックの37型FullHDのαIPS液晶テレビがついに壊れました。

15年も使ってきました。昨年あたりから時々電源が一時的に落ちるような症状があったので、電源周りのコンデンサーが劣化したのではないかと思います。

一度修理しようと分解しかけたんですが、バックパネルを外すことが出来ずに断念しました。

ということでメインモニターがなくなったので、サブモニターをメインにして使っています。

2021年の夏に購入した物ですが、まだ廃番にされていないようです。

27インチ、解像度WQHD(2560*1440)、リフレッシュレート170Hz、HDR400対応、色域sRGB比130%/DCI-P3 95%、IPS液晶というゲーミングディスプレイとしてはなかなかハイスペックな仕様です。

ただ37インチFullHDをメインとして使っていたので、27インチWQHDだと字がかなり小さいです。

ドットピッチが60dpiから108dpiになったので当たり前ですが。

で、サブモニターにもう一台同じ物を注文しました。

どこのショップも在庫が無いようで、結構探しました。

他のディスプレイでも良いように思いますが、このディスプレイ以外だと色味の違いが大きくなりそうなので、敢えて同じ物にしました。

色が違うと結構なストレスになるので。

ゆくゆくはモニターアームを買って自由に動かせるようにしたいです。
