# yuki9902::blog
## 概要 (Overview)

yuki9902.comとして、1999年から2013年まで公開していた記事の一部をサルベージして、アメブロ「[yuki9902::blog](https://ameblo.jp/yuki9902/)」で2009年から現在までに公開している記事を纏めて掲載しています。

現在でも、yuki9902.comを参照しているサイトが存在しているので役に立てば嬉しく思います。

As yuki9902.com, I have salvaged some articles originally published from 1999 to 2013 and compiled them here alongside articles published on the Ameba Blog "[yuki9902::blog](https://ameblo.jp/yuki9902/)" from 2009 to the present.

Since some websites still reference yuki9902.com, I hope this collection proves useful.
